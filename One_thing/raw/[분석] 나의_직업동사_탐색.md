# 나의 직업 동사 탐색

> 직업탐색-동사법 프레임워크 적용
> raw 폴더 + wiki 폴더 전체 분석 기반

---

## 분석 전 확인 — 이 탐색의 목적

"나는 뭘 좋아하지?"가 아니라 **"나는 뭘 해도 안 힘들지?"**를 찾는 것.
도파민적 설렘이 아니라 고통의 역치로 찾는다.

---

## 1단계 — 반복적으로 나타나는 동사 수집

raw 폴더 전체에서 이 사람이 반복적으로 하는 행동을 추출했다.
설명이나 결심이 아니라 **실제로 한 것들**.

### 관찰 기록

| 동사 | 등장한 파일 / 증거 |
|---|---|
| **분석하다** | 악플러 심리 분석, 팩트풀니스 언론비판, 정치 비판, 심리학고양이 레퍼런스 분석, 수요비탄력성 분석 |
| **연결하다** | "새벽 담배 피우다 오토바이 보고 생각이 연결됨", 데미안→소설집필 연결, 팩트풀니스→정치 연결, 수많은 wiki 교차참조 |
| **글로 쏟다** | [생각쏟기] 태그 파일 다수. 기쁠 때도 쓰고, 화날 때도 쓰고, 절망할 때도 씀 |
| **질문하다** | "왜 저러는 거야", "왜 이렇게 됐을까", "이게 뭘까", 모든 관찰이 질문으로 이어짐 |
| **설명하다 / 번역하다** | 어려운 개념을 일상 언어로 바꾸려는 시도가 전체 raw에서 반복됨 |
| **시스템 만들다** | PARA 시스템, 옵시디언 위키, 쇼핑몰 자동화, 크롤링 아이디어 |
| **비판하다 / 꿰뚫다** | 진보 위선 비판, 언론 구조 비판, 시스템 비판 — 본질을 보려는 충동 |

## 2단계 — 고통의 역치 테스트

**고통이 아닌 것들 (낮은 역치):**

- 10년 넘게 반복 단순 노동을 버텼다 → **몸이 힘든 것은 참는다**
- 새벽 4시반에도 생각하고 연결하고 있다 → **생각 자체가 멈추지 않는다**
- 화났을 때도 "왜 저러는 거지"로 분석이 먼저 나온다 → **분석이 자동반사다**
- 절망적인 상황에서도 글로 쓴다 ([마인드] 자기연민과_사랑.md) → **글쓰기가 배출구다**
- 아이디어가 떠오르면 연결이 자동으로 일어난다 → **연결이 의도적이지 않다**
- 게임에서 몇 시간씩 반복 사냥 → **반복적 집중 가능**
- 독서 후 반드시 현실에 대입하려 한다 → **적용하지 않으면 읽은 게 아님이라는 기준**

**고통인 것들 (높은 역치):**

- 시간의 자유가 없는 것 → 이걸 못 참아서 일용직 탈출 계획을 세웠다
- 시스템 안에 갇혀 대체 가능한 것 → 핵심 동기
- **앞에서 직접 말하지 못하는 것** → "앞에선 말 못하고 자기 전에 머릿속으로 참교육 상상을 많이 함"

## 3단계 — 자기 기록 탐색

### 시간을 쓰는 곳
- 독서 + 독서 후 사색 + 옵시디언 정리
- 생각 쏟아내기 (글쓰기)
- 아이디어 연결 및 시스템 설계

### 자연스럽게 클릭한 콘텐츠
- 고명환 (독서 → 현실 적용), 심리학고양이 (표본 넓은 주제 + 책 연결), 갓냥이 (팩트 + 논리 + 직설), 자청 (표본이론, 구조 분석)

### 신나서 떠드는 주제
- 사람들이 왜 저런 행동을 하는지
- 시스템이 왜 이렇게 돌아가는지
- 사업 아이디어 구조

## 4단계 — 핵심 동사 5개 추출

raw + wiki 전체에서 반복적으로, 의식하지 않아도 자동으로, 힘들지 않게 하는 동사들:

```
관찰한다
    ↓
분석한다 (왜 그런지 파고든다)
    ↓
연결한다 (전혀 다른 영역과 이어붙인다)
    ↓
설명한다 (쉬운 말로 번역한다)
    ↓
쓴다 (글로 쏟아낸다)
```

이 다섯 개가 **이 사람이 살아있는 동사들**이다.

## 5단계 — 이 동사들로 가능한 직업

동사가 맞으면 어떤 포맷이든 작동한다. 중요한 건 포맷이 아니라 동사다.

| 직업 | 핵심 동사 매핑 | 고통 역치 |
|---|---|---|
| 유튜브 크리에이터 (대본형) | 관찰+분석+연결+설명+쓰다 | 낮음 |
| 뉴스레터 / 서브스택 | 분석+연결+설명+쓰다 | 매우 낮음 |
| 작가 (논픽션/에세이) | 관찰+분석+연결+쓰다 | 낮음 |
| 칼럼니스트 | 분석+비판+쓰다 | 낮음 |
| 유튜브 크리에이터 (직접 출연) | 분석+설명+**말하다** | **말하다가 검증 안 됨** |

---

## 6단계 — 중요한 발견 하나

"앞에선 말 못 하고 자기 전에 머릿속으로 참교육 상상을 많이 함"

이게 직접 출연 유튜브에서 고통 역치가 높을 수 있다는 신호다.
갓냥이 포맷 (AI 음성)을 레퍼런스로 선택한 건 취향이 아니라 고통 역치 판단이 이미 작동한 거다.

**대본을 쓰는 것 = 낮은 역치**
**대본을 읽어서 직접 말하는 것 = 아직 검증 안 됨**

---

## 7단계 — 정체성 자본 점검

지금까지 쌓인 것들이 어떻게 연결되는가:

```
10년 일용직 현장 경험
    → 시스템 구조가 몸으로 각인됨
    → "대체 가능한 톱니바퀴"라는 자각
    
독서 시작
    → 세상이 다르게 보임
    → 분석이 심화됨
    
분노에서 연민으로의 변화
    → "빌런 참교육" → "왜 저러는지 이해"
    → 이 변화 자체가 콘텐츠가 됨
    
옵시디언 위키 구축
    → 생각을 시스템으로 만드는 능력
    → 지식이 흩어지지 않고 연결됨
```

이 경로 전체가 정체성 자본이다. 아무것도 버려진 게 없다.

## 결론

**이 사람의 직업 동사: 관찰-분석-연결-설명-쓰다**

이 동사들이 전부 작동하는 형태는:

1. **글 중심 유튜브 채널** (대본 씀 → AI 음성 또는 자막 영상) → 가장 자연스러운 출발점
2. **뉴스레터 / 브런치** → 직접 말 안 해도 됨. 글만으로 먼저 시작 가능
3. **소설 / 에세이** → "망치를 든 작가" 욕구와 정렬. 장기 목표

**가장 현실적인 순서:**
글 쓰기 → 글로 만든 대본 → 영상화

직접 얼굴 노출이나 목소리를 먼저 결정할 필요 없다.
대본이 좋으면 포맷은 나중에 골라도 된다.

## 반례 / 한계

- 분석하고 설명하고 싶은 욕구가 강하지만, 분야가 아직 확정되지 않음 (정치 비판, 심리학, 자기계발, 사업 분석 — 전부 흥미 있음)
- "직업"이 아니라 "수익"을 원한다고 느낀 파일들이 많음 → 동사보다 결과(돈/자유)가 목표일 수 있음 → 이건 문제가 아님, 동사와 목표가 정렬되어 있는지 계속 확인할 것
- 소설 쓰기 욕구가 반복해서 나옴 → 아직 실제로 해보지 않음 → 동사 고통 역치 미검증
