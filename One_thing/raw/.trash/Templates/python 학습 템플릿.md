## 3️⃣ 실습 문제 리플레이 (백지 코딩 3회 반복) ⭐

강의에서 풀었던 실습 문제를 **코드를 완전히 지운 상태**에서 다시 풉니다.
(새로운 영상의 조언대로, **최소 3번** 반복하여 손이 문법을 기억하게 만듭니다.)

| 실습 번호          | 1회차 (기억 인출) | 2회차 (속도/숙달) | 3회차 (주석 설명) |
| :------------- | :---------- | :---------- | :---------- |
| **문제 6** (강아지) | [ ] 성공      | [ ] 성공      | [ ] 성공      |
| **문제 7** (합계)  | [ ] 성공      | [ ] 성공      | [ ] 성공      |
| ...            | [ ]         | [ ]         | [ ]         |


> **💡 오답 노트 / 막힌 부분:**
> * *문제 7번에서 합계 변수를 for문 안쪽에 만들어서 계속 0으로 초기화되는 실수를 했다. 변수는 바깥에 만들어야 한다!*


> **💡 3회차 풀이 중 느낀 점 / 응용 아이디어:**
> * (예: 강아지 이름을 사용자 입력으로 받아보면 어떨까?)
