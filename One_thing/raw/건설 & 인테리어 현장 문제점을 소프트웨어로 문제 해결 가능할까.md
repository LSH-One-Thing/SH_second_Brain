## 🔨 건설/인테리어 현장 특화 소프트웨어 아이디어 20선

### 1. **공정 간섭 알림 시스템**

- 📌 문제: 다른 작업 공정이 겹쳐 충돌 발생 (예: 도배 전에 전기공사 끝나야 함)
    
- ✅ 해결: 공정 순서 자동화 → 간섭 가능성 사전 경고
    
- 🛠️ 기능: 공정 등록 → 순서 충돌 시 경고 표시
    

---

### 2. **모바일 자재 재고 관리 시스템**

- 📌 문제: 자재 떨어진 후에야 인지, 공정 지연
    
- ✅ 해결: 자재 수량 알림 및 자동 발주 알림
    
- 🛠️ 기능: 최소 수량 도달 시 알림, QR 코드로 입출고 관리
    

---

### 3. **현장용 '시공 동영상 가이드' 시스템**

- 📌 문제: 미숙련 작업자가 시공 방식 모름 → 하자 발생
    
- ✅ 해결: 작업별 짧은 영상 가이드 제공
    
- 🛠️ 기능: QR 코드 찍으면 영상 시청 가능 (도배, 전기, 배관 등)
    

---

### 4. **현장 내 작업자 실시간 위치 확인**

- 📌 문제: 작업자 찾기 어려움, 어디서 작업 중인지 모름
    
- ✅ 해결: 스마트폰 GPS 기반 위치 공유
    
- 🛠️ 기능: 관리자 화면에서 실시간 위치 보기 (프라이버시 고려 필요)
    

---

### 5. **작업 일지 자동 생성 도구**

- 📌 문제: 작업 일지를 수기로 작성 → 귀찮고 누락
    
- ✅ 해결: 사진 + 음성만으로 일지 자동 생성
    
- 🛠️ 기능: “사진 찍고 → 음성 녹음 → 텍스트 변환 → 일지 완성”
    

---

### 6. **하자보수 요청 자동화 앱**

- 📌 문제: 고객이나 현장관리자가 하자 접수하면 전달 누락 많음
    
- ✅ 해결: 하자 접수 → 자동 분류 → 담당자에게 알림
    
- 🛠️ 기능: 사진 첨부, 위치 선택, 카카오 알림톡 연동
    

---

### 7. **도면 수정 추적 시스템**

- 📌 문제: 도면 자주 바뀌는데, 최신 도면 공유 안 됨
    
- ✅ 해결: 버전 관리 + 주석 기록 시스템
    
- 🛠️ 기능: 도면 업로드 → 변경 이력 저장 → 버전 관리
    

---

### 8. **현장 안전 체크 자동화 시스템**

- 📌 문제: 매일 해야 하는 안전 점검을 수기로 처리
    
- ✅ 해결: 체크리스트 기반 자동 기록 + 위반 시 경고
    
- 🛠️ 기능: QR 체크인 + 작업별 위험 요소 확인
    

---

### 9. **인건비 정산 자동 시스템 (일용직용)**

- 📌 문제: 엑셀 수기로 정산 → 실수 많고 귀찮음
    
- ✅ 해결: 출퇴근 + 시급 입력만으로 정산서 출력
    
- 🛠️ 기능: 근무일수, 야근, 주휴 등 자동 계산 → PDF 출력
    

---

### 10. **건설업 특화 회계·세무 자동화 툴**

- 📌 문제: 소사장들이 세무를 잘 모름
    
- ✅ 해결: 건설업 맞춤형 세무관리 자동화
    
- 🛠️ 기능: 매출/매입 입력 → 부가세 신고용 엑셀 자동 생성
    

---

### 11. **사진 기반 측정 앱 (AI/AR 활용)**

- 📌 문제: 줄자 없이 측정해야 할 때 많음
    
- ✅ 해결: 사진 찍으면 길이 자동 추정
    
- 🛠️ 기능: AI 기반 거리 측정, AR 기반 시뮬레이션
    

---

### 12. **3D 인테리어 시뮬레이터 (모바일용)**

- 📌 문제: 고객이 인테리어 결과 상상이 어려움
    
- ✅ 해결: 방 사진 찍고 가상 인테리어 시뮬레이션 제공
    
- 🛠️ 기능: 벽지/바닥/가구 배치 시뮬레이션
    

---

### 13. **현장 일정 공유 캘린더**

- 📌 문제: 팀원/하청업체 간 일정 공유가 안 됨
    
- ✅ 해결: 작업 일정 공유 캘린더 제공
    
- 🛠️ 기능: 캘린더 + 담당자 태그 + 공정 연결
    

---

### 14. **소음/진동 측정기 연동 앱**

- 📌 문제: 소음 민원 자주 발생
    
- ✅ 해결: 센서와 연동하여 데이터 시각화 및 기록
    
- 🛠️ 기능: 소음/진동 초과시 SMS 자동 전송
    

---

### 15. **클라이언트 피드백 수집 시스템**

- 📌 문제: 공사 완료 후 고객 불만 파악이 늦음
    
- ✅ 해결: 피드백 설문 자동 발송 + 분석
    
- 🛠️ 기능: 템플릿 설문 → 자동 통계 분석 제공
    

---

### 16. **하청 계약서 자동 생성기**

- 📌 문제: 하청 계약서를 손으로 작성 → 누락 위험
    
- ✅ 해결: 조건 입력 → 계약서 자동 생성
    
- 🛠️ 기능: 단가, 기간, 조건 입력 → 계약서 PDF 출력
    

---

### 17. **현장 소통용 전용 메신저 (기록 중심)**

- 📌 문제: 카톡으로 소통하다가 나중에 내용 분실
    
- ✅ 해결: 작업 별 대화방 + 대화 내용 자동 저장
    
- 🛠️ 기능: 이슈 기록, 사진 첨부, @멘션 기능
    

---

### 18. **계측기/장비 대여·반납 관리 시스템**

- 📌 문제: 장비 대여 후 분실/반납 누락 자주 발생
    
- ✅ 해결: QR 코드 찍고 대여/반납 기록 남김
    
- 🛠️ 기능: 사용자/일시/상태 자동 로그 저장
    

---

### 19. **건설 관련 법규/기준 자동 검색 앱**

- 📌 문제: 시공 기준이나 법규를 즉석에서 찾기 어려움
    
- ✅ 해결: 키워드 기반 건설 규정 검색
    
- 🛠️ 기능: "방수 기준", "최소 천장 높이" 검색 → 요약 제공
    

---

### 20. **공정 단가 및 실적 비교 분석 툴**

- 📌 문제: 실적 대비 수익 파악 어려움
    
- ✅ 해결: 예상 vs 실제 단가 자동 비교
    
- 🛠️ 기능: 공사별 인건비/자재비 분석 → 원가 관리
