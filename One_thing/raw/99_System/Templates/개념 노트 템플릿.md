```markdonw
# [개념] getattr 
## 1. 뭐하는 녀석인가? (한 줄 요약) 
> 객체 안에 특정 속성이 있는지 확인하고, 없으면 에러 대신 
> 내가 정한 기본값(False 등)을 주는 안전장치.
```


```Python
# sys 모듈에 'frozen'이라는 속성이 있으면 그 값을, 없으면 False를 반환 
if getattr(sys, 'frozen', False): 
	print("포장된 exe 파일입니다.")
```
