> **이 파일의 용도**
> 채널을 시작하거나 흔들릴 때마다 돌아오는 기준점. 모든 콘텐츠 결정은 이 한 장으로 판단한다.

---

## 채널 한 줄 정의

> **"세상의 큰 사건을 인간 본능으로 해석하고, 개인의 일상으로 착지시키는 채널"**

---

## 채널 철학 (왜 존재하는가)

```
세상을 움직이는 것은 이념이 아니라 인간의 본능이다.
전쟁도, 주식도, 관계도 — 결국 같은 회로에서 나온다.
이 구조를 알면 당하지 않고 선택할 수 있다.
```

**핵심 메시지 3개 (영상마다 하나를 반드시 담는다):**

1. "당신이 못 나서가 아니라 게임이 이렇게 설계되어 있다"
2. "뇌가 그렇게 반응하도록 되어 있다 — 당신 혼자만 이런 게 아니다"
3. "메커니즘을 알면 다르게 선택할 수 있다"

---

## 핵심 타겟 — 단 한 명으로 정의

**이 한 사람에게 말한다:**

> 뉴스를 봐도 표면만 보인다. 미국이 전쟁을 하고, 주식이 폭락하고, 사람들이 왜 저렇게 행동하는지 — 원인을 이념이나 운으로만 설명하는 게 찜찜하다. 더 깊은 구조가 있을 것 같은데 어디서 찾을지 모른다. 세상이 원래 이렇게 설계된 거라면 나는 어떻게 해야 하는가.

**공통 특징:**
- 호기심이 많고 표면적 설명에 만족 못 한다
- "왜?"를 자주 묻는 사람
- 당모리를 보다가 "그래서 내 삶이랑 뭔 상관인데?"라는 생각을 한 번쯤 했다
- 뉴스를 보지만 구조가 안 보인다
- 연령대: 20대 후반~40대 / 성별 무관 / 생각이 많은 사람

**타겟이 아닌 사람:**
- 단순 정보 요약을 원하는 사람
- 명상/힐링 콘텐츠를 원하는 사람
- 빠른 팁만 원하는 사람

## 주제 기둥 4개

모든 영상은 이 4개 중 하나에 속한다. 섞이거나 넘어가면 채널 정체성이 흔들린다.

### 기둥 1 — 거시 사건 × 인간 본능 [메인]

```
입구: 전쟁, 외교, 국제 사건, 사회 통계
출구: 그 사건을 만든 인간의 본능 + 내 일상에서도 작동하는 방식
```

예시 제목:
- "미국이 70년 동안 중동에서 실패하는 진짜 이유"
- "러시아가 우크라이나를 침공할 수밖에 없었던 뇌과학"
- "일본이 사죄를 못 하는 이유 — 국가도 자의식 방어기제가 있다"

### 기둥 2 — 경제/사회 구조 × 인간 심리 [서브 메인]

```
입구: 주식 데이터, 결혼 통계, 소비 패턴, 산업 구조
출구: 이 구조가 인간 심리를 어떻게 설계했는가
```

예시 제목:
- "358만 계좌가 증명한 것 — 열심히 할수록 망하는 이유"
- "갓생의 배신 — WHO 데이터로 보는 완벽주의의 뇌과학"
- "40대 여성이 결혼 못 하는 진짜 이유 — 진화심리학 분석"

### 기둥 3 — 뇌과학/심리 × 반직관 [접근성 높은 입구]

```
입구: 누구나 경험한 현상 (가족에게 폭발, 번아웃, 손절 못함)
출구: 뇌과학 메커니즘 → 설계로 접근하는 방법
```

예시 제목:
- "가족에게만 괴물이 되는 이유 — 뇌과학 4가지 분석"
- "집에 오면 폭발하는 당신이 나쁜 사람이 아닌 이유"
- "손절 못 하는 이유 — 손실회피가 주식 계좌를 갉아먹는 구조"

### 기둥 4 — 역사/인물 × 인간 본능 패턴 [깊이 콘텐츠]

```
입구: 역사 사건, 위인 또는 악인의 행동
출구: 반복되는 인간 본능 패턴 → 오늘의 세상과 연결
```

예시 제목:
- "히틀러가 독일 국민을 사로잡을 수 있었던 이유"
- "왜 모든 제국은 같은 방식으로 무너지는가"
- "에릭호퍼가 말한 것 — 혁명가와 악플러는 같은 심리다"

## 포맷

| 항목 | 결정 |
|---|---|
| 얼굴 노출 | 없음 (나레이션 전용) |
| 메인 포맷 | 롱폼 10~20분 (신뢰 자산) |
| 서브 포맷 | 숏츠 60초 이하 (미끼) |
| 영상 구조 | 거시 충격 → 상식 뒤집기 → 본능 메커니즘 → 개인 착지 |
| 톤 | 직설적·냉정하되 따뜻함 있음. "당신이 나쁜 게 아니라 이런 구조다" |
| 시리즈 | 하나의 주제를 3~5편으로 쪼개는 시리즈 설계 |

**영상 구조 공식:**
```
0:00~0:30  — 충격적인 거시 현상 제시 (클릭한 사람의 기대 확인)
0:30~2:00  — "근데 왜 그럴까" 질문 + 상식과의 충돌
2:00~8:00  — 구조 분석 (인간 본능 메커니즘으로 설명)
8:00~10:00 — 개인 삶으로 착지 ("당신의 일상에서도 이렇게 작동한다")
마무리      — "이 메커니즘을 알면 어떻게 선택할지는 당신의 몫"
```

## 경쟁 채널과의 포지셔닝

| 채널 유형 | 설명 방식 | 이 채널과의 차이 |
|---|---|---|
| 당모리 (역사/국제) | 사실·데이터·역사 → 지식 전달 | 같은 입구, 다른 출구. 이 채널은 인간 본능으로 설명하고 개인 삶으로 착지 |
| 뇌부자들/바른심리학 | 개인 심리 현상 → 뇌과학 설명 | 이 채널은 거시 사건을 먼저 입구로 쓴다. 표본이 더 크다 |
| 갓냥이 | 팩트+논리+직설 / 정치 비판 | 비판이 아니라 분석. 정치 편향 없음 |
| 자청 | 자기계발·성공 | 이 채널은 권위가 아니라 메커니즘으로 설명한다 |

**이 채널이 비어 있는 이유:**
- 거시 입구 + 인간 본능 렌즈 + 개인 착지 = 이 조합으로 일관되게 운영하는 채널이 없다
- 당모리는 "왜 이 사건이 일어났나"에서 끝난다
- 심리학 채널은 개인 현상에서 시작한다
- 이 채널은 둘 사이의 다리다

## 하지 않는 것 (금지 목록)

이것을 어기면 채널 정체성이 무너진다.

- ❌ 권위 없는 자기계발 — "이렇게 살면 성공합니다" 형식
- ❌ 이념 편향 — 특정 정치 진영 편들기
- ❌ 단순 정보 나열 — "오늘은 [개념]에 대해 설명합니다" 형식
- ❌ 해결책이 없는 불만 토로 — "세상이 이래서 문제다"로 끝나는 영상
- ❌ 이론 먼저 현상 나중 — "손실회피란 무엇인가" 형식 (개념 설명 채널이 아님)
- ❌ 표본이 좁은 주제 — 특정 직업군·연령대만 공감하는 소재

## 채널 운영 판단 기준 (매 영상 체크리스트)

영상 기획 전 이 3가지를 확인한다:

```
1. 이 주제에 진짜 에너지가 있는가?
   (에너지가 없으면 만들어도 안 나온다)

2. 거시 입구가 있는가?
   (전 국민 표본 소재인가, 아니면 특정 집단만의 이야기인가)

3. 개인 착지가 있는가?
   ("그래서 내 삶이랑 뭔 상관인데?"에 답이 있는가)
```

셋 다 없으면 만들지 않는다.

## 첫 번째 영상 후보 3개

| 후보 | 기둥 | 근거 |
|---|---|---|
| A. "미국이 70년간 중동에서 실패하는 진짜 이유" | 기둥 1 | 진짜 에너지 있음 + 전 국민 표본 + 위키 재료 풍부 (패권경쟁, 집단정체성, 손실회피) |
| B. "갓생의 배신 — 열심히 살수록 뇌가 망가지는 이유" | 기둥 2 | 표본 넓음 (번아웃 경험자 전체) + WHO 데이터 거시 입구 + 개인 공감 강함 |
| C. "358만 계좌가 증명한 것 — 주식에서 열심히 할수록 망하는 이유" | 기둥 2 | 데이터 충격적 + 반직관 강함 + 위키 재료 완비 |

**첫 영상 선택 원리:** 가장 에너지가 강하게 느껴지는 쪽. 에너지가 화면에 나온다.

## 3년 후 이 채널의 모습

```
구독자 10만+ (롱폼 신뢰 채널)
책 1권 (위키 → 원고 → 출간)
강의/컨설팅 (메커니즘 이해 기반 의사결정)
```

당모리가 80만에서 책을 냈듯이 — 위키의 36개 페이지는 이미 책의 원자재다.
