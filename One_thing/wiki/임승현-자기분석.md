## 망치를 든 작가 — 삶의 사명

> "니체를 망치를 든 철학자라고 말한다. 나는 망치를 든 작가가 되고 싶다."

구체적 내용: 단순히 인생을 나열하는 방식이 아니라, 사회적 문제에 대해 독자가 한 번쯤 생각할 수 있는 글을 쓰고 싶다. 본질을 보지 못하는 사람들에게 본질을 볼 수 있도록 해주는 것.

이것이 콘텐츠를 "단순 정보 전달"이 아닌 "구조 폭로"로 설계하는 이유와 같다.

## 핵심 동사 5개

**분석 방법**: "나는 뭘 좋아하지?"가 아니라 "나는 뭘 해도 안 힘들지?"

```
관찰한다 (위화감을 흘려보내지 않고 붙잡는다)
    ↓
분석한다 (왜 그런지 비합리적 의사결정 구조로 파고든다)
    ↓
연결한다 (행동경제학 / 심리학 / 진화심리학 / 신경과학 / 철학을 가로지른다)
    ↓
설명한다 (이론 인용이 아니라 일상 언어로 번역한다)
    ↓
쓴다 (글로 쏟아낸다 — 발행은 아직 미해결 병목)
```

**이 5개 동사가 자동 작동한 실제 증거:**
- 화났을 때: 악플러 심리 분석이 먼저 나옴
- 절망적일 때: 짜장면 사건 직후 "이거 콘텐츠 각인데?"가 먼저 나옴
- 컨디션 안 좋을 때: 산책길 커플 대화를 분석
- 정치적으로 거슬릴 때: 공소취소 특검법 구조 분석

→ **조건 없이 작동하는 동사가 진짜 동사다.**

## INTP 구조와 결과 지향성

**INTP이지만 결과 지향적:**
- "눈에 띄는 결과물"을 좋아한다
- 난간대 설치처럼 갯수가 정해져 있거나 결과물이 보이는 것을 선호
- 분석만 하고 발행을 못 하는 패턴의 핵심 원인이 여기 있다: 결과물이 안 보이는 상태를 오래 견디지 못함

**강점:**
- 분석적 사고 — 사안을 구조와 원리로 분해하는 것이 자연스럽다
- 크로스 도메인 연결 — 심리학·경제학·진화론·철학을 교차해 하나의 설명을 만드는 것에 쾌감
- 시스템 지향 — 반복 가능한 구조를 선호
- 독립·익명 선호 — 얼굴 노출, 교사 포지셔닝, 고접촉 포맷을 피한다
- 자기 가설이 틀렸을 때 인정하고 뒤집기 가능 (INFP→INTP 수정 이력)

**구조적 약점 (결함이 아닌 설계 문제):**

| 약점 | 구체적 증거 |
|---|---|
| 강한 분석 인풋, 약한 실행 사이클 | 솔로 프로젝트(쇼츠, 정치분석, 카드뉴스)가 매번 같은 지점에서 멈춤 |
| 외부 피드백 없이 확신 유지 불가 | 6개월 이상 무반응 구간을 솔로로 통과한 이력 없음 |
| 완성 안 된 결과물 공개 못함 | 분석은 검증됐는데 발행이 안 되는 패턴 반복 |
| 앞에서 직접 말하기 회피 | "앞에선 말 못 하고 자기 전에 참교육 상상 많이 함" |

→ 대응 전략: 첫 콘텐츠는 "7번째 버전의 가장 거친 초안"으로 프레임. 발행 자체가 외부 압력 구조가 되도록 공개 약속 박아두기.

1. **자원 희소가 능력을 강제한다** — 자본 없이 성공하면 무기가 두 개: 실력 + 절약 능력
2. **현금흐름 구조가 유리한 시장만 진입** — 제조업·건설업·선불 비용·재고·미수금 구조는 배제
3. **검증된 모델 복사가 올바른 진입점** — 차별화 포인트 하나만 추가
4. **집행 전 산업 간 통찰 밀도가 확보돼야 한다** — 이해 없이 속도는 의미 없다
5. **솔로 확신 소멸이 진짜 실패 모드** — 방법론 문제가 아니라 외부 피드백 부재 구조 문제

- V-Score: 본질강화×2.5 + 재구매율×4.5 + 확장성×3.5 / 원가×2.5 + 재고×1 + 유통비×2 + 경쟁강도×4
- C-Score: 조회수×3 + 전환율×4 + 객단가×3 + 반복성×2.5 / 제작비×2 + 제작시간×2 + 경쟁강도×3 + 수명리스크×2

## 의외의 연결 — 이 위키가 증명하는 것들

**1. 톱니바퀴 이론 ↔ 자청 9단계 상승나선**
- 건설 현장에서 배운 통찰("일용직이든 사장이든 시스템 안에 갇혀 있다")이 자청의 9단계 이론과 정확히 맞닿는다
- 9단계를 올라가는 것 = 대체 불가능한 대장 톱니바퀴가 되는 것. 같은 목표의 다른 언어.

**2. 망치를 든 작가 ↔ 데미안 ↔ 빅터 프랭클**
- 데미안: "알을 깨고 날아오르는 것" = 자신만의 세계관으로 기존을 부수는 것
- 프랭클: "어떤 태도를 취할 것인가"라는 마지막 자유
- 망치를 든 작가: 본질을 못 보는 사람들에게 본질을 보게 해주는 것
- 세 개 모두 "자기만의 렌즈로 세상을 부수고 재건한다"는 구조를 공유

**3. 짜장면 사건 ↔ 자의식-방어기제의 반대 작동**
- 자의식 방어기제: 모욕이나 상처를 받으면 자기 보호를 위해 정보를 차단하거나 왜곡
- 임승현: 현장에서 짜장면 맞은 직후 "이거 콘텐츠 각인데?"가 먼저 나옴
- 이것은 자의식 방어기제가 작동하지 않고 오히려 분석·서사화로 전환되는 패턴
- 위화감을 흘려보내지 않고 붙잡는 핵심 동사 "관찰하다"의 증거

**4. 돈 이후에도 할 것 = 독서·글쓰기·소설 ↔ 직업탐색-동사법**
- 동사법의 핵심 질문: "돈이 없어도, 돈이 많아도 하는 것?"
- 임승현이 "매달 1000억씩 입금돼도" 할 것으로 꼽은 것들: 독서, 글쓰기, 유튜브, 소설
- 이 4개가 전부 핵심 동사 5개(관찰·분석·연결·설명·쓰다)와 정확히 매핑됨
- 검증 통과.

**5. INTP 결과 지향성 ↔ 유창성-착각-메타인지의 역설**
- 유창성 착각: "이해한 것 같다" = 실제로는 아무것도 남지 않은 상태
- 임승현은 결과물(눈에 보이는 것)을 좋아하는 INTP
- 분석은 방대하게 하지만 "발행"이라는 눈에 보이는 결과가 없으면 자신의 작업이 실재하는지 확신을 잃는다
- → 발행이 단순히 콘텐츠 배포가 아니라 "내가 실제로 존재하고 있음"을 확인하는 행위일 수 있다

**6. 이 위키 시스템 자체 ↔ UIS 창조방법론**
- U: 지식이 흩어지고 연결이 안 되는 불편함
- I: Obsidian 기반 위키 시스템
- S: 같은 불편함을 가진 지식 수집가들
- 위키의 40개 페이지 = 콘텐츠 40개 이상의 원자재
