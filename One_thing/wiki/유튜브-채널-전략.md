> **핵심 Takeaway (콘텐츠 방향성 관점)**
> 채널의 핵심 명제: **세상을 움직이는 것은 이념이 아니라 인간의 본능이다.** 전쟁·외교·역사의 큰 사건을 진화심리학·행동경제학 렌즈로 해석하고 개인 삶의 패턴으로 착지시킨다. 입구는 세계 뉴스, 출구는 자기 이해. 이론에 매몰되지 않고 거시 현상에서 출발해 인간 본능으로 설명하는 것이 핵심이다.

---

## 유튜브 채널 전략

### 채널 정체성

**철학:**
> "세상을 움직이는 것은 이념이 아니라 인간의 본능이다. 전쟁, 외교, 역사의 큰 사건도, 개인의 선택도 — 결국 같은 회로에서 나온다. 이 구조를 알면 당하지 않고 선택할 수 있다."

**정체성:** 거시 사건을 인간 본능으로 해석하는 분석가  
**미션:** 세상의 큰 사건과 개인의 행동 패턴이 같은 뿌리에서 온다는 것을 보여준다

**핵심 가치 제안:**
- 시청자(Win): "유용한 콘텐츠"로 욕구를 해결하고 생각하는 힘을 얻는다
- 채널(Win): 신뢰도 기반의 지식 콘텐츠 수익 모델

---

### 시청자 욕망 지도

콘텐츠가 건드려야 할 7가지 핵심 욕망:

| 욕망 유형 | 구체 표현 | 키워드 |
|---|---|---|
| **명확한 해석** | 세상이 왜 이렇게 돌아가는가 — 전쟁·외교·정치가 내 삶에 미치는 영향 | 국제정치, 역사, 시사 해설 |
| **지적 각성** | 복잡한 세상을 꿰뚫는 관점, 남들이 모르는 구조를 안다 | 인문, 철학, 역사, 행동경제학 |
| **방어/간파** | 속임수 간파, 이 게임의 규칙을 안다 | 심리학, 협상, 구조 분석 |
| **자기 이해** | 나는 왜 이렇게 행동하는가 — 개인 패턴이 세상과 연결됨 | 심리학, 행동경제학 |
| **경제적 생존** | 미래 불확실성에서 살아남기 | 투자, 경제 구조 |

---

### 콘텐츠 제작 4단계

#### 1단계: 욕망 탐색 (주제 발굴)
"좋은 콘텐츠는 사람들의 대체적인 욕망을 건드려야 한다"

실행 방법:
- 서점 베스트셀러 분야 관찰
- 커뮤니티 불만 글의 주제 파악
- 성공한 영화/드라마의 공통 주제 의식 분석
- 조회수 높은 유사 콘텐츠의 [[콘텐츠-쪼개기-분석법]] 적용

#### 2단계: 컨셉 확정
욕망을 바탕으로 채널 컨셉과 구조 확정.

#### 3단계: 콘텐츠 기획 (차별화)
**핵심 원칙:** 책이 아니라 **"질문/목적"을 중심**으로 구성

```
X 잘못된 접근: "오늘은 [동물농장]을 소개합니다"
O 올바른 접근: "왜 우리는 스스로 생각하지 않고 조종당할까?
              이 답을 찾기 위해 [동물농장], [1984], [길 위의 철학자]를 
              엮어보았습니다"
```

#### 4단계: 스크립트 작성
- 시청자가 지적 각성을 얻고 스스로 생각할 수 있도록 유도
- 논리 + 감정 자극 + 실용적 통찰의 결합

### 콘텐츠 흐름 — 거시에서 개인으로

채널의 핵심 구조:

```
거시 사건 (전쟁/외교/역사/사회)    ← 입구: 세계 뉴스, 표본 극대
        ↓
"왜 저렇게 행동하는가?"           ← 질문: 상식 뒤집기
        ↓
진화/심리/행동경제학 메커니즘      ← 설명: 위키 개념 적용
        ↓
"개인 삶에서도 동일하게 작동"     ← 착지: 시청자 자기 이해
```

예시:
- "미국이 관세전쟁을 하는 이유" → [[손실회피-편향]] × [[집단정체성-내외집단]] → 개인 협상에서도 같은 패턴
- "러시아가 우크라이나를 침공한 이유" → [[에릭호퍼-좌절한-이상주의자]] × [[손실회피-편향]] → 개인이 잃어버린 것을 되찾으려 할 때의 심리
- "제국이 반드시 무너지는 이유" → [[클루지-진화의-유산]] → 조직이 커지면 왜 망가지는지

이 구조가 당모리와 다른 이유: 당모리는 역사적 사실로 설명하고 끝난다. 이 채널은 인간 본능으로 설명하고 개인 삶으로 착지한다.

---

### 콘텐츠 구성 원칙

**행동경제학/심리학 적용 시 주의사항:**

```
X 잘못된 흐름: 이론 → 사례 찾기
O 올바른 흐름: 구체 현상 → 이론으로 설명
```

예:
- X: "손실 회피 편향이란 무엇인가?" → 검색해도 조회수 낮음 (낯설고 어려움)
- O: "개미들이 손절 못 하는 진짜 이유" → 친숙한 현상 → 손실 회피 편향으로 설명

**이론에 매몰되지 않는 방법:**
- AI에게 "이걸 행동경제학으로 풀어줘"라고 묻지 않는다 → 매몰된다
- 대신 "이 스크립트 구조 분석해줘", "이 표현 더 자연스럽게 바꿔줘"라고 기능 요청으로만 쓴다
- 이론 확인은 한 줄짜리 검증만: "이게 손실 회피 맞아?" → 그 이상 X

### 채널 포맷 옵션

| 포맷 | 특징 | 적합 주제 |
|---|---|---|
| 쇼츠 (1분 이하) | 빠른 현상 분석, 후킹 우선 | 사회 현상, 심리 원리 요약 |
| 롱폼 (10-20분) | 심층 분석, 지적 만족 | 책 연계, 사회구조 분석 |
| 에세이형 | 1인칭 경험 + 분석 | 현장 원자료, 자기 성찰 |
| AI 음성 활용 | 갓냥이형 — 팩트+논리 전달 | 정치/시사 비평 |

---

### 비즈니스 모델 단계

1. **초기 (지금):** 조회수 수익 목표 — 1개라도 업로드
2. **중기:** 광고주 컨택 — "이 구독자 수준에서 이 내용으로 광고 가능합니다" 제안서
3. **후기:** 제3 상품 — 전자책, 강의, 컨설팅으로 채널을 레퍼러로 활용

---

### 행동 원칙

1. 조회수 잘 나온 레퍼런스 먼저 찾기 → 구조 분석 → 원리 추출
2. 초안은 내가 러프하게 → AI는 기능 요청으로만 활용
3. 이론 용어보다 일상 언어로 카피라이팅
4. 1개 올리고 피드백 → 수정 → 반복
