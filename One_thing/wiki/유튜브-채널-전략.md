> **핵심 Takeaway (콘텐츠 방향성 관점)**
> 채널의 철학은 하나다: "생각하는 힘을 기르는 것이 국가와 개인 모두를 발전시킨다." 하지만 철학이 아무리 좋아도 시청자의 뇌가 클릭하지 않으면 의미없다. 심리학/행동경제학 원리를 콘텐츠 구조에 통합하고, 이론에 매몰되지 않고 구체적 일상 현상에서 출발하는 것이 핵심이다.

---

## 유튜브 채널 전략

### 채널 정체성

**철학:**
> "생각하는 힘을 길러야 국가는 발전할 수 있다. 무지함과 냉소적인 태도의 사람들이 많다면 나라는 기득권을 가진 사람들 손바닥에서 놀아날 수밖에 없다."

**정체성:** 비판적 이상주의자  
**미션:** 개인의 "지적 각성"을 돕고, 사회의 "집단 지성"을 높인다

**핵심 가치 제안:**
- 시청자(Win): "유용한 콘텐츠"로 욕구를 해결하고 생각하는 힘을 얻는다
- 채널(Win): 신뢰도 기반의 지식 콘텐츠 수익 모델

---

### 시청자 욕망 지도

콘텐츠가 건드려야 할 7가지 핵심 욕망:

| 욕망 유형 | 구체 표현 | 키워드 |
|---|---|---|
| 경제적 생존 | 미래 불확실성에서 살아남기 | 투자, 재정 자유, 트렌드 |
| 방어/간파 | 속임수 간파, 자기 보호 | 다크 심리학, 손자병법 |
| 명확한 해석 | 정치/사회 변화가 내 삶에 미치는 영향 | 시사 해설 |
| 나만의 무기 | 즉시 써먹을 수 있는 실용 기술 | N잡, AI 활용 |
| 지적 각성 | 복잡한 세상을 꿰뚫는 관점 | 인문, 철학, 역사 |
| 관계 피로 해소 | 불필요한 마찰 없이 관계 유지 | 화술, 말투 |
| 자기 이해 | 나는 왜 이렇게 행동하는가 | 심리학, 행동경제학 |

---

### 콘텐츠 제작 4단계

#### 1단계: 욕망 탐색 (주제 발굴)
"좋은 콘텐츠는 사람들의 대체적인 욕망을 건드려야 한다"

실행 방법:
- 서점 베스트셀러 분야 관찰
- 커뮤니티 불만 글의 주제 파악
- 성공한 영화/드라마의 공통 주제 의식 분석
- 조회수 높은 유사 콘텐츠의 [[콘텐츠-쪼개기-분석법]] 적용

#### 2단계: 컨셉 확정
욕망을 바탕으로 채널 컨셉과 구조 확정.

#### 3단계: 콘텐츠 기획 (차별화)
**핵심 원칙:** 책이 아니라 **"질문/목적"을 중심**으로 구성

```
X 잘못된 접근: "오늘은 [동물농장]을 소개합니다"
O 올바른 접근: "왜 우리는 스스로 생각하지 않고 조종당할까?
              이 답을 찾기 위해 [동물농장], [1984], [길 위의 철학자]를 
              엮어보았습니다"
```

#### 4단계: 스크립트 작성
- 시청자가 지적 각성을 얻고 스스로 생각할 수 있도록 유도
- 논리 + 감정 자극 + 실용적 통찰의 결합

---

### 콘텐츠 구성 원칙

**행동경제학/심리학 적용 시 주의사항:**

```
X 잘못된 흐름: 이론 → 사례 찾기
O 올바른 흐름: 구체 현상 → 이론으로 설명
```

예:
- X: "손실 회피 편향이란 무엇인가?" → 검색해도 조회수 낮음 (낯설고 어려움)
- O: "개미들이 손절 못 하는 진짜 이유" → 친숙한 현상 → 손실 회피 편향으로 설명

**이론에 매몰되지 않는 방법:**
- AI에게 "이걸 행동경제학으로 풀어줘"라고 묻지 않는다 → 매몰된다
- 대신 "이 스크립트 구조 분석해줘", "이 표현 더 자연스럽게 바꿔줘"라고 기능 요청으로만 쓴다
- 이론 확인은 한 줄짜리 검증만: "이게 손실 회피 맞아?" → 그 이상 X

### 채널 포맷 옵션

| 포맷 | 특징 | 적합 주제 |
|---|---|---|
| 쇼츠 (1분 이하) | 빠른 현상 분석, 후킹 우선 | 사회 현상, 심리 원리 요약 |
| 롱폼 (10-20분) | 심층 분석, 지적 만족 | 책 연계, 사회구조 분석 |
| 에세이형 | 1인칭 경험 + 분석 | 현장 원자료, 자기 성찰 |
| AI 음성 활용 | 갓냥이형 — 팩트+논리 전달 | 정치/시사 비평 |

---

### 비즈니스 모델 단계

1. **초기 (지금):** 조회수 수익 목표 — 1개라도 업로드
2. **중기:** 광고주 컨택 — "이 구독자 수준에서 이 내용으로 광고 가능합니다" 제안서
3. **후기:** 제3 상품 — 전자책, 강의, 컨설팅으로 채널을 레퍼러로 활용

---

### 행동 원칙

1. 조회수 잘 나온 레퍼런스 먼저 찾기 → 구조 분석 → 원리 추출
2. 초안은 내가 러프하게 → AI는 기능 요청으로만 활용
3. 이론 용어보다 일상 언어로 카피라이팅
4. 1개 올리고 피드백 → 수정 → 반복
