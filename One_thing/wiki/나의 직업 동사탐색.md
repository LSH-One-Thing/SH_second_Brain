> **핵심 Takeaway (콘텐츠 방향성 관점)** 
> 이 사람의 핵심 동사는 관찰-분석-연결-설명-쓰다. 이 다섯 개가 기쁠 때도, 화날 때도, 절망할 때도 자동으로 작동했다. 직업을 고르는 기준은 "좋아하는가"가 아니라 "이 동사들이 작동하는 환경인가"다.

---

## 나의 직업 동사 탐색

### 분석 방법

[[직업탐색-동사법]]의 프레임워크를 적용했다. "나는 뭘 좋아하지?"가 아니라 **"나는 뭘 해도 안 힘들지?"**

도파민적 설렘이 아니라 고통의 역치로 직업을 찾는다. 재능이란 결과물을 잘 만드는 것이 아니라, **결과물을 만들기까지의 과정을 남들보다 덜 힘들게 견디는 힘**이다.

---

### 반복적으로 나타난 동사들

raw 파일 전체에서 설명이나 결심이 아니라 **실제로 반복한 것들**만 추출했다.

|동사|실제 증거|
|---|---|
|**분석하다**|악플러 심리 분석, 언론 구조 분석, 정치 비판, 레퍼런스 채널 해부 — 화났을 때도 분석이 먼저 나옴|
|**연결하다**|"새벽 담배 피우다 오토바이 보고 생각이 연결됨" — 의도하지 않아도 자동으로 일어남|
|**글로 쏟다**|[생각쏟기] 태그 파일 다수. 기쁠 때도, 화날 때도, 절망할 때도 글로 씀|
|**질문하다**|모든 관찰이 "왜 저러는 거지"로 이어짐. 분석 전에 질문이 먼저 나옴|
|**설명하다**|어려운 개념을 일상 언어로 바꾸려는 시도가 전체 raw에서 반복됨|
|**시스템 만들다**|PARA, 옵시디언 위키, 자동화 아이디어 — 구조를 만드는 것|

### 고통의 역치 테스트

**낮은 역치 (안 힘든 것들):**

- 10년 넘게 건설 현장 중기공·곰방을 버텼다 → 몸이 힘든 것은 견디지만, "축적되지 않는 반복"으로 분류해서 탈출 대상으로 명확히 지정함
- 짜장면 사건 직후에도 "이거 콘텐츠 각인데?"가 먼저 나왔다 → 현장에서 받은 모욕마저 분석·서사 재료로 변환되는 반사
- 한 개념을 외부 데이터로 매번 의심하는 부담 → 자청 책의 주장, 도파민의 역설, 에고 고갈 이론을 직접 반박 검증한 이력
- 자기 가설이 틀렸을 때 인정하고 뒤집기 → INFP→INTP 자기 진단 수정, "내 가설이 쾌락 쪽으로 기울어 있었다" 즉시 인정한 패턴
- 결론이 안 나오는 모호한 상태에 오래 머무르기 → 진로 방향을 몇 개월에 걸쳐 가설-검증으로 굴림
- 시스템 내부 작동 원리를 바닥까지 파고들기 → 코딩은 "원리부터", 영어는 어원, 선물거래는 펀딩비·OI·청산맵까지 한 달씩 잡고 학습
- 한 시스템의 비효율 지점을 끈질기게 찾기 → 레퍼런스 유튜버 프로세스를 보자마자 "어디 줄일 수 있나"로 사고 이동
- 추상을 구체 사례로 번역하기 → 행동경제학 개념을 트레이딩 심리·콘텐츠 전략·정치 분석에 즉시 매핑
- 글을 여러 번 고쳐 쓰기 → 초고→검토 패스 2단계 워크플로우를 본인이 설계
- 혼자 머리 쓰는 고독, 의자에 오래 앉아 있기 → 본인이 "의자에 앉아서 RPG 하는 거 안 고통스럽다"고 직접 언급

**높은 역치 (힘든 것들):**

- 같은 결과만 나오는 단순 반복 노동 → 건설업 10년이 정확히 이 카테고리, 본인이 탈출 대상으로 명시
- 자본·현금흐름이 불리한 일을 계속하는 것 → 제조·재고·플랫폼·수주업을 사업 후보에서 명시적으로 배제 (핵심 원칙 ②)
- 위계와 눈치로 작동하는 조직 → 인력사무실 단가 협상, 현장 꼴통과의 멱살 사건에서 반복 확인
- 감정 기반 설득·영업, 큰 그룹 앞 즉흥 발언 → 콘텐츠 방향에서 일관되게 회피
- **외부 피드백 없이 혼자 확신을 유지하는 상태** → 핵심 병목. 솔로 프로젝트(쇼츠, 정치 분석, 카드뉴스)가 매번 이 지점에서 멈춤. 핵심 원칙 ⑤와 정확히 일치
- **완성 안 된 결과물을 세상에 공개하는 심리적 노출** → 분석은 검증됐는데 발행이 안 되는 패턴이 반복됨
- "준비만 계속하고 시작 못 하는 상태" → 본인이 직접 "가장 큰 실수였다"고 인정
- 외상 매출·미수금이 발생하는 구조 → 건설 현장에서 학습된 회피 영역
- 얼굴 노출·퍼스널 브랜딩·"가르치는 척" → 메모리에 명시적 회피 항목으로 고정
- **앞에서 직접 말하는 것** → "앞에선 말 못 하고 자기 전에 참교육 상상 많이 함"

### 자기 기록 탐색 결과

**시간을 쓰는 곳:** 독서 + 사색 + 옵시디언 정리 / 생각 글로 쏟기 / 아이디어 연결 / 거상 시장 분석·트레이딩 학습 (단, 거상은 콘텐츠 소재로 쓰지 않음)

**자연스럽게 클릭한 채널:** 고명환(독서→현실 적용), 심리학고양이(표본 넓은 주제+책 연결), 자청(구조 분석)

**신나서 떠드는 주제:** 사람들이 왜 비합리적으로 행동하는가 / 시스템이 왜 이렇게 돌아가는가 / 정치인은 왜 같은 무리수를 반복하는가 / 갈라치기는 왜 작동하는가

**검증된 반응 데이터:** 정서불안햄찌 댓글 1,200 좋아요, 심리학고양이 댓글 414 좋아요. 두 번 다 "권위 있는 학자의 해설"이 아니라 **"비슷한 상황에 있는 사람이 위화감을 잡아내고 풀어쓴 글"**에서 나온 반응. 즉 검증된 건 학위가 아니라 **관찰의 날카로움 + 솔직한 1인칭 서술**.

### 핵심 동사 5개

```
관찰한다 (위화감을 흘려보내지 않고 붙잡는다)
    ↓
분석한다 (왜 그런지 비합리적 의사결정 구조로 파고든다)
    ↓
연결한다 (행동경제학 / 심리학 / 진화심리학 / 신경과학 / 철학을 가로지른다)
    ↓
설명한다 (이론 인용이 아니라 일상 언어로 번역한다)
    ↓
쓴다 (글로 쏟아낸다, 발행은 미해결 영역)
```

이 다섯 개는 화났을 때(악플러 분석), 절망적일 때(현장에서 짜장면 맞은 직후), 컨디션 안 좋을 때(산책길 커플 대화), 정치적으로 거슬릴 때(공소취소 특검법) 전부 자동으로 작동했다.

### 동사로 본 직업 가능성

동사가 맞으면 어떤 포맷이든 작동한다. 포맷보다 동사가 먼저다. 단, **다섯 번째 동사 "쓴다"의 마지막 단계인 "발행"이 모든 솔로 프로젝트의 병목**이라는 점을 무시하면 안 된다.

|직업|동사 매핑|고통 역치|병목|
|---|---|---|---|
|익명 뉴스레터·브런치 (비합리적 의사결정 구조 분석)|관찰+분석+연결+설명+쓰다|매우 낮음|발행|
|유튜브 크리에이터 (대본형, AI 음성, 위화감 포착 포맷)|관찰+분석+연결+설명+쓰다|낮음|발행 + 편집 허들|
|픽션 (단편 소설, 짜장면 장면 계열)|관찰+분석+연결+쓰다|**부분 검증** (단편 작성·구조 분석 진행 중)|발행 + 양가성 견디기|
|자동화·Apps Script 외주|시스템 만들다|낮음|영업·대인 일부|
|유튜브 (직접 출연)|분석+설명+**말하다**|**미검증** (앞에서 말하기 회피 명시)|검증 자체 안 됨|

### 정체성 자본 점검

지금까지 쌓인 것들이 어떻게 연결되는가:

```
10년 건설 현장 경험
    → 시스템 구조가 몸으로 각인됨 + 현금흐름 미스매치 구조를 직접 겪음
    → "대체 가능한 톱니바퀴" 자각 + 사업 필터(자본·재고·미수금 회피)의 근거
    → 짜장면 사건 같은 1차 원자료 — 픽션의 그릇 재료

독서 + 옵시디언 두 층 노트 시스템
    → 행동경제학·심리학·진화심리학·신경과학·철학을 교차시키는 분석 도구
    → "한 원리로 여러 영역을 동시에 푸는" 선호 사고 패턴

분노에서 위화감 포착으로의 변화
    → 빌런 참교육 욕구 → "왜 저렇게 행동할 수밖에 없는 구조인가"로 전환
    → 양가성을 견디는 능력으로 발전 (혼모노 분석에서 본인이 자각)
    → 이 변화 자체가 콘텐츠 서사가 됨

검증된 반응 (좋아요 1.2k / 414)
    → 학위 없이도 "관찰의 날카로움"만으로 공명이 일어난다는 데이터
    → 권위 기반 해설 채널이 아니라 위화감 포착 포맷이 본인 동사와 정렬

레퍼런스 분석 자산
    → 장항준 4단계를 혼모노에 적용하는 독서·재구성 방법론
    → 용의자 X의 헌신 구조 분석
    → 도치추리(howcatchem) 장르 매핑
```

자동화 작업(거상 스크래퍼, 가족 쇼핑몰)은 정체성 자본에 포함하되, **완성·운영 중인 독립 프로젝트로 분류하지 않는다.** AI 보조로 만든 학습 흔적이지 포트폴리오가 아니다.

거상 게임은 시장 감각의 사적 학습 도구로만 쓰고, 콘텐츠·분석 사례로는 쓰지 않는다.

### 결론 — 가장 자연스러운 경로

**위화감 포착 → 비합리적 의사결정 구조로 분석 → 일상 언어로 번역 → 익명 발행**

얼굴·목소리·"가르치는 척"을 먼저 결정할 필요가 없다. 권위·학위로 무장한 이론 해설 포맷이 아니라, **"이거 이상한데?"를 5초 안에 보여주는 1인칭 위화감 포맷**이 검증된 반응 데이터와 정렬된다.

단, 이 경로는 **6개월 무반응 구간을 솔로로 통과해야 한다.** 본인의 다섯 번째 핵심 원칙(외부 피드백 부재 시 솔로 프로젝트 멈춤)과 정면으로 충돌하는 구간이다. 동사가 맞아도 구조가 안 받쳐주면 또 멈춘다.

대응:

- 발행 단위를 "잘 쓴 글"이 아니라 "7번째 버전의 가장 거친 초고"로 정의
- 익명 채널이라도 발행 자체가 외부 압력 구조가 되도록 공개 약속을 어딘가에 박아둠
- 첫 5편은 주제 선정·확장성·수익화를 일절 고려하지 않음 — 발행 행위 자체만 측정

### 미검증·부분검증 동사

**부분검증 — 픽션 쓰기** 짜장면 사건 단편 초고, 2031년 풍자 소설 바이블, 도치추리 구조 분석은 진행됐다. 하지만 단편 한 편이라도 발행까지 가본 적은 없다. 양가성을 끝까지 유지하는 기법(혼모노 5번째 원칙)이 본인 정서와 정렬되는지도 5편을 써본 후에야 역산 가능하다.

**미검증 — 직접 말하기·얼굴 노출** 구조적으로 회피 영역이라 검증 시도 자체가 비효율. 다른 동사들이 먼저 자산화된 후에나 재검토할 영역.

**미검증 — 외부 피드백 구조 없는 장기 솔로 발행** 6개월 이상 무반응 구간을 견딘 적이 한 번도 없다. 이건 의지의 문제가 아니라 구조의 문제로 분류해야 한다.

### 반례 / 한계

- 분석 분야가 "비합리적 의사결정 구조"로 좁혀졌지만, 진입 그릇(소재)이 정치/심리/사기/노동/픽션 중 어디인지는 아직 발행 데이터로 검증되지 않음
- "직업"보다 "월 200~300만원 안정 현금흐름"이 실질 목표 → 동사와 수익 구조가 정렬되는 포맷(익명 뉴스레터+자동화 외주)이 콘텐츠 단독보다 현실적
- 고통의 역치만으로 보면 자학적 접근이 됨. 핵심 원칙 ④(밀도 있는 통찰이 실행에 선행) + ⑤(외부 피드백 구조)와 함께 봐야 함
- "쓴다"의 마지막 단계가 검증 안 된 상태에서 모든 분석이 의미 있다고 결론 내리면 같은 함정이 반복됨
