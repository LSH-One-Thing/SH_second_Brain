> **핵심 Takeaway (콘텐츠 방향성 관점)**
> 이 사람의 핵심 동사는 관찰-분석-연결-설명-쓰다. 이 다섯 개가 기쁠 때도, 화날 때도, 절망할 때도 자동으로 작동했다. 직업을 고르는 기준은 "좋아하는가"가 아니라 "이 동사들이 작동하는 환경인가"다.

---

## 나의 직업 동사 탐색

### 분석 방법

[[직업탐색-동사법]]의 프레임워크를 적용했다.
"나는 뭘 좋아하지?"가 아니라 **"나는 뭘 해도 안 힘들지?"**

도파민적 설렘이 아니라 고통의 역치로 직업을 찾는다.
재능이란 결과물을 잘 만드는 것이 아니라, **결과물을 만들기까지의 과정을 남들보다 덜 힘들게 견디는 힘**이다.

---

### 반복적으로 나타난 동사들

raw 파일 전체에서 설명이나 결심이 아니라 **실제로 반복한 것들**만 추출했다.

| 동사 | 실제 증거 |
|---|---|
| **분석하다** | 악플러 심리 분석, 언론 구조 분석, 정치 비판, 레퍼런스 채널 해부 — 화났을 때도 분석이 먼저 나옴 |
| **연결하다** | "새벽 담배 피우다 오토바이 보고 생각이 연결됨" — 의도하지 않아도 자동으로 일어남 |
| **글로 쏟다** | [생각쏟기] 태그 파일 다수. 기쁠 때도, 화날 때도, 절망할 때도 글로 씀 |
| **질문하다** | 모든 관찰이 "왜 저러는 거지"로 이어짐. 분석 전에 질문이 먼저 나옴 |
| **설명하다** | 어려운 개념을 일상 언어로 바꾸려는 시도가 전체 raw에서 반복됨 |
| **시스템 만들다** | PARA, 옵시디언 위키, 자동화 아이디어 — 구조를 만드는 것 |

---

### 고통의 역치 테스트

**낮은 역치 (안 힘든 것들):**

- 10년 넘게 단순 반복 노동을 버텼다 → 몸이 힘든 것은 참는다
- 새벽 4시반에도 생각이 연결되고 있다 → 생각이 멈추지 않는다
- 화났을 때도 분석이 자동으로 나온다 → 분석이 반사다
- 절망적인 상황에서도 글로 쓴다 → 글쓰기가 배출구다
- 아이디어가 떠오르면 연결이 자동으로 일어난다 → 연결이 의도적이지 않다

**높은 역치 (힘든 것들):**

- 시간의 자유가 없는 것 → 일용직 탈출 계획의 핵심 이유
- 시스템 안에 갇혀 대체 가능한 것 → 핵심 동기
- **앞에서 직접 말하는 것** → "앞에선 말 못 하고 자기 전에 참교육 상상을 많이 함"

---

### 자기 기록 탐색 결과

**시간을 쓰는 곳:**
독서 + 사색 + 옵시디언 정리 / 생각 글로 쏟기 / 아이디어 연결

**자연스럽게 클릭한 채널:**
고명환(독서→현실 적용), 심리학고양이(표본 넓은 주제+책 연결), 갓냥이(팩트+논리+직설), 자청(구조 분석)

**신나서 떠드는 주제:**
사람들이 왜 저런 행동을 하는지 / 시스템이 왜 이렇게 돌아가는지 / 사업 아이디어 구조

---

### 핵심 동사 5개

```
관찰한다
    ↓
분석한다 (왜 그런지 파고든다)
    ↓
연결한다 (전혀 다른 영역과 이어붙인다)
    ↓
설명한다 (쉬운 말로 번역한다)
    ↓
쓴다 (글로 쏟아낸다)
```

이 다섯 개가 이 사람이 살아있는 동사들이다.

---

### 동사로 본 직업 가능성

동사가 맞으면 어떤 포맷이든 작동한다. 포맷보다 동사가 먼저다.

| 직업 | 동사 매핑 | 고통 역치 |
|---|---|---|
| 유튜브 크리에이터 (대본형, AI 음성) | 관찰+분석+연결+설명+쓰다 | 낮음 |
| 뉴스레터 / 브런치 | 분석+연결+설명+쓰다 | 매우 낮음 |
| 작가 (논픽션/에세이) | 관찰+분석+연결+쓰다 | 낮음 |
| 소설가 | 관찰+분석+연결+쓰다 | **미검증** |
| 유튜브 (직접 출연) | 분석+설명+**말하다** | **미검증** |

---

### 중요한 발견 — 갓냥이 포맷 선택의 의미

갓냥이 포맷(AI 음성)을 레퍼런스로 고른 것은 취향이 아니다.

"앞에선 말 못 하고 자기 전에 머릿속으로 참교육 상상을 많이 함"이라는 기록이 있다.
→ 직접 말하기는 아직 고통 역치가 검증되지 않았다.
→ 반면 대본을 **쓰는 것**은 고통 역치가 명확히 낮다.

```
대본을 쓰는 것 → 낮은 역치 (확인됨)
대본을 말로 읽는 것 → 미검증
```

---

### 정체성 자본 점검

지금까지 쌓인 것들이 어떻게 연결되는가:

```
10년 일용직 현장 경험
    → 시스템 구조가 몸으로 각인됨
    → "대체 가능한 톱니바퀴" 자각 → 핵심 콘텐츠 주제

독서 시작
    → 세상이 다르게 보임
    → 분석이 심화됨

분노에서 연민으로의 변화
    → 빌런 참교육 하고 싶었다 → 왜 저러는지 이해
    → 이 변화 자체가 콘텐츠 서사가 됨

옵시디언 위키 구축
    → 생각을 시스템으로 만드는 능력
    → 지식이 흩어지지 않고 연결됨
```

아무것도 버려진 게 없다. 전부 정체성 자본으로 쌓여 있다.

---

### 결론 — 가장 자연스러운 경로

**글 쓰기 → 글로 만든 대본 → 영상화**

얼굴이나 목소리를 먼저 결정하지 않아도 된다.
대본이 좋으면 포맷은 나중에 골라도 된다.

---

### 미검증 동사 — 소설 쓰기

소설 쓰고 싶다는 욕구가 raw 파일 여러 곳에서 반복됐다.
단, 아직 실제로 써본 적이 없다. 고통 역치 미검증 상태다.
→ 짧은 글이라도 한 번 써봐야 진짜 동사인지 알 수 있다.

---

### 반례 / 한계

- 분석하고 싶은 분야가 아직 확정되지 않음 (정치 비판, 심리학, 자기계발, 사업 분석 — 전부 흥미 있음)
- "직업" 보다 "수익"이 목표인 파일들이 많음 → 동사와 수익 목표가 정렬되어 있는지 계속 확인 필요
- 고통의 역치만이 기준이 되면 너무 자학적 접근이 됨. 동기와 의미도 함께 봐야 함
