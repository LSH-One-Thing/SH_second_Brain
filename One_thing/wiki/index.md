## 의외의 연결 지도 (Cross-Domain Connections)

가장 충격적인 교차 연결 — 다른 도메인에서 동일 원리가 독립적으로 발견된 것들:

| 연결 | 핵심 통찰 |
|---|---|
| [[전략적-대충-역설]] ↔ [[개미 투자자가 실패하는 이유]] | 심리학 채널과 투자 채널이 완전히 다른 경로로 "아무것도 안 하는 것이 최고"라는 동일 결론 도달 |
| [[행복최적화-역설]] ↔ [[전략적-대충-역설]] ↔ [[개미 투자자가 실패하는 이유]] | 행복 심리학 + 투자 + 뇌과학 세 도메인이 독립적으로 "강도 추구 → 역효과"라는 동일 구조 발견 — 이 채널만의 연결 |
| [[가까운 사람에게 화를 더 잘 내는 이유]] ↔ [[에릭호퍼-좌절한-이상주의자]] | 가족에 대한 분노 ↔ 광신자의 내부 배신자 혐오: 기대치가 높은 대상에게 실망이 폭발하는 동일 구조 |
| [[개미 투자자가 실패하는 이유]] ↔ [[더닝-크루거-효과]] | 가장 많이 공부한 20대가 꼴찌: 지식이 자신감을 높이지만 구조적 불리함을 이기지 못한다 |
| [[패권경쟁-심리학]] ↔ [[가까운 사람에게 화를 더 잘 내는 이유]] | 국가 간 동맹 배신 ↔ 가족 폭발: 기대치와 애착이 클수록 배신의 충격이 크다 |
| [[전략적-대충-역설]] ↔ [[편도체-강탈]] | 완벽주의 강박 = 편도체 강탈: 뇌에서 동일하게 처리됨 |
| [[거시-마이크로-연결법]] ↔ 위키 전체 | 이 위키의 모든 개념이 거시-마이크로 연결의 브리지 재료다 |
| [[에릭호퍼-좌절한-이상주의자]] ↔ [[더닝-크루거-효과]] | 좌절한 이상주의자 = 더닝-크루거(무지의 산) + 자의식 방어의 결합체. "내가 못 나서가 아니라 시스템이 나빠서" = 능력 부족을 인식 못한 채 외부 귀인하는 구조 |
| [[이야기-구조-언더독]] ↔ [[에릭호퍼-좌절한-이상주의자]] | 혁명·대중 운동 = 집단 단위 언더독 서사. 선동가는 항상 "우리가 약자·피해자" 프레이밍을 쓴다. 스토리텔링과 정치 심리학이 같은 회로를 쓴다 |
| [[신뢰-프리미엄-비탄력성]] ↔ [[집단정체성-내외집단]] | 충성 구독자 = 내집단 형성. 브랜드 충성도와 종교 충성도와 팬덤이 같은 원리 — 신뢰 채널은 사실 내집단을 만드는 것이다 |
| **[[자청-시스템1-원시인뇌]] ↔ [[클루지-진화의-유산]] ↔ [[편도체-강탈]]** | **3자 수렴 (충격):** 카네만(심리학), 자청(사업), 마커스(신경과학)가 완전히 독립적으로 같은 결론에 도달 — "인간은 원시 뇌에 지배당한다." 시스템1 = 클루지 = 편도체 강탈 = 원시인 뇌. 같은 현상의 세 가지 언어 |
| **[[자청-본능분석-반박제거]] ↔ [[에릭호퍼-좌절한-이상주의자]]** | **마케터와 혁명가의 동일 공식:** 에릭호퍼 "좌절한 대중의 본능(자기 실패 외부 귀인) 파악 + 반박 제거(당신이 나쁜 게 아니야)"가 자청의 "구매자 본능 파악 + 반박 제거"와 동일 구조. 선동 = 마케팅 |
| **[[자청-본능분석-반박제거]] ↔ [[자의식-방어기제]] ↔ [[확증편향]]** | **가치 사다리의 심리학적 기반:** 가치 사다리가 작동하는 이유 = "내가 이미 투자했다 = 틀리지 않았다"(자의식 방어) + 초기 긍정 인상이 이후 정보를 긍정적으로 필터링(확증편향). 마케팅이 인지 편향을 의도적으로 설계하는 구조 |
| **[[자청-UIS-창조방법론]] ↔ [[직업탐색-동사법]]** | **불편함이 직업이다:** UIS의 U(불편함)와 동사법 탐색의 출발점("내가 자주 고통받는 것")이 동일. 사업 아이디어와 직업 발견이 같은 방법으로 찾아진다 |
