# 자청의 쪼개기 이론 — 유튜브 적용법

> 출처: 자청 유튜브 / 쪼개기 이론 분석
> 작성일: 2026-05-20

---

## 쪼개기 이론이란?

자청의 쪼개기 이론은 **하나의 넓은 주제를 점점 좁고 구체적인 하위 주제로 분해**해 나가는 콘텐츠 전략이다.

핵심 전제:
- 넓은 주제 = 경쟁 과다 + 시청자 이탈률 높음 + 알고리즘 불리
- 좁은 주제 = 타깃 명확 + 이탈률 낮음 + 검색 노출 유리 + 전문성 인식

**쪼개기의 3가지 축:**
1. **대상 쪼개기** — "사람들" → "20대 직장인 남성" → "첫 직장 6개월 차"
2. **상황 쪼개기** — "스트레스" → "퇴근 후 스트레스" → "상사에게 무시당한 날 저녁"
3. **원인 쪼개기** — "왜 실패하는가" → "왜 시작은 하는가" → "왜 3일 후에 멈추는가"

## 쪼개기 작동 원리 (구조도)

```
[대주제]
  └── [중주제 A]
        └── [소주제 A-1]  ← 영상 1개
        └── [소주제 A-2]  ← 영상 1개
  └── [중주제 B]
        └── [소주제 B-1]  ← 영상 1개
        └── [소주제 B-2]  ← 영상 1개
```

대주제 하나 = 영상 수십 개의 원천
각 소주제 영상이 대주제로 연결되는 내부 링크 구조 → 채널 체류 시간 증가

## 실전 예시 — 주제: "부자가 되지 못하는 이유"

> 이 주제를 선택한 이유: 보편적 공감대 + 인간 심리·본능으로 설명 가능 + 유사 채널 포화 상태이므로 쪼개기 효과가 극명하게 드러남

### 0단계: 대주제 (업로드 불가 수준)
```
"부자가 되는 법"
→ 너무 넓다. 경쟁자 수천 개. 검색해도 상위 노출 불가.
```

### 1단계: 중주제 쪼개기 (카테고리 정의)
```
부자가 되지 못하는 이유
  ├── A. 돈을 버는 문제
  ├── B. 돈을 쓰는 문제  ← 심리·본능으로 설명하기 가장 좋음
  ├── C. 돈을 모으는 문제
  └── D. 돈에 대한 인식 문제
```

### 2단계: 소주제 쪼개기 (영상 후보)

**B. 돈을 쓰는 문제** 를 쪼갠다:

| 소주제 | 영상 제목 예시 | 연결 심리 원리 |
|--------|----------------|----------------|
| B-1 | 월급이 올랐는데 왜 저축이 안 늘어날까 | 파킨슨 법칙, 쾌락 적응 |
| B-2 | 왜 우리는 세일할 때만 필요 없는 걸 사는가 | 손실 회피, 앵커링 |
| B-3 | 왜 카드 긁을 때는 아무 생각이 없다가 명세서 보면 놀라는가 | 고통 지연 효과 |
| B-4 | 왜 가난할수록 복권을 더 많이 사는가 | 희망 편향, 가능성 과대평가 |
| B-5 | 친구가 명품 사면 나도 사고 싶어지는 이유 | 사회적 비교, 지위재 심리 |

### 3단계: 초소주제 쪼개기 (알고리즘 최적화)

**B-2 "왜 세일할 때만 필요 없는 걸 사는가"** 를 다시 쪼갠다:

```
B-2-a: "50% 세일 표시만 봐도 심장이 뛰는 과학적 이유"
B-2-b: "블랙프라이데이가 당신을 멍청하게 만드는 방법"
B-2-c: "쇼핑몰이 '마감 임박'을 쓰는 진짜 이유"
B-2-d: "당신이 할인 쿠폰 때문에 더 많이 쓰는 이유"
```

→ 같은 원리(손실 회피 + 앵커링)를 **맥락만 바꿔** 4개 영상 생산 가능

## 쪼개기 할 때 자주 쓰는 프레임 5가지

### 1. "알면서도 왜?" 프레임
```
일반: "다이어트 방법"
쪼개기: "살을 빼야 한다는 걸 알면서도 못 하는 이유"
→ 지식-행동 간극을 파고든다. 공감 + 죄책감 자극.
```

### 2. "특정 순간" 프레임
```
일반: "화를 잘 내는 이유"
쪼개기: "누군가 내 앞에서 새치기할 때 유독 폭발하는 이유"
→ 시청자가 자신의 구체적 경험과 즉시 매칭.
```

### 3. "반직관" 프레임
```
일반: "절약하는 법"
쪼개기: "절약을 열심히 할수록 오히려 더 쓰게 되는 이유"
→ 예상 반전 → 클릭 유도.
```

### 4. "집단 비교" 프레임
```
일반: "성공한 사람의 특징"
쪼개기: "왜 똑같이 노력했는데 A는 성공하고 B는 실패했는가"
→ 비교 구조가 서사를 만든다.
```

### 5. "원인의 원인" 프레임
```
일반: "왜 사람들은 거짓말을 하는가"
쪼개기: "왜 들킬 걸 알면서도 거짓말을 멈추지 못하는가"
→ 표면 원인 한 겹 더 파고들기. 심리 채널에 특히 유효.
```

## 쪼개기 이론의 핵심 법칙 요약

| 원칙 | 설명 |
|------|------|
| **1개 영상 = 1개 핵심 원리** | 한 영상에서 원리 2개 이상 → 시청자 혼란 → 이탈 |
| **제목이 곧 쪼개기의 완성도** | 제목이 구체적일수록 쪼개기가 잘 된 것 |
| **쪼갤수록 시리즈가 된다** | 소주제들이 연결되면 자동으로 플레이리스트 → 체류 시간 증가 |
| **쪼개기 전 검색 수요 확인** | 아무리 좁아도 아무도 안 찾으면 무의미 (키워드 도구 필수) |
| **대주제는 채널 정체성** | 소주제 영상들이 쌓이면 대주제 검색에서도 노출되기 시작 |

## 내 채널에 적용한다면

채널 대주제: **"인간이 왜 비합리적으로 행동하는가"**

쪼개기 예시:
```
인간의 비합리적 행동
  ├── 돈 앞에서 바보가 되는 이유 (행동경제학)
  │     ├── 왜 손해를 안 보려다가 더 큰 손해를 보는가 (손실 회피)
  │     └── 왜 비싼 것이 더 좋아 보이는가 (가격 = 품질 휴리스틱)
  ├── 관계에서 바보가 되는 이유 (사회심리학)
  │     ├── 왜 나쁜 사람인 걸 알면서도 연락을 끊지 못하는가
  │     └── 왜 칭찬보다 한 마디 비난이 더 오래 기억되는가
  └── 선택 앞에서 바보가 되는 이유 (인지심리학)
        ├── 왜 선택지가 많을수록 아무것도 못 고르는가
        └── 왜 이미 쓴 돈 때문에 더 나쁜 결정을 하는가
```

## 참고: 쪼개기 실패 패턴

- **너무 좁아진 경우**: "30대 중반 이직 3번 경험 있는 남성 직장인이 퇴근 후 충동 구매를 하는 이유" → 검색 수요 없음
- **쪼개기 없이 넓은 주제**: "심리학" → 유튜브 상위 노출 불가, 채널 정체성 없음
- **쪼개기만 하고 연결 안 한 경우**: 각 영상이 독립적이면 채널 체류 시간 안 늘어남 → 내부 링크·카드 필수
