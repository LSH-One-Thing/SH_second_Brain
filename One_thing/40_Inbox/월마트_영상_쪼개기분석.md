# 월마트 영상 쪼개기 분석

> 대상: 지식한입 "시총 11위 먹은 월마트의 비결" (2026-05-09)
> 목적: 이 영상이 왜 잘됐는지 나노 단위로 쪼개서 공식 추출
> 작성일: 2026-05-20

---

## 분석 전 전제

쪼개기의 목표 = **"왜 이게 잘됐지?"를 답할 수 있는 공식 추출**
공식이 추출되면 → 내 채널에 이식 가능

---

## 1. 제목 쪼개기

```
"시총 11위 먹은 월마트의 비결"
```

| 요소 | 내용 | 기능 |
|------|------|------|
| "먹은" | 의인화 + 공격적 동사 | 중립적 "달성한"보다 생동감, 클릭욕 자극 |
| "11위" | 구체적 숫자 | 막연한 "상위권"보다 신뢰감 + 궁금증 |
| "비결" | 숨겨진 정보 암시 | "내가 모르는 게 있다"는 정보 격차 자극 |

**추출 공식:**
> `[의외의 주체]` + `[공격적/의인화 동사]` + `[숨겨진 이유]`
> 예: "삼성이 애플한테 진 진짜 이유" / "이마트가 쿠팡을 못 이기는 구조"

## 2. 오프닝 훅 쪼개기 (0:00~1:35)

### 실제 구조

```
① 충격 팩트 투하
   "1조 달러 기업이 전 세계에 딱 13개"

② 이분법으로 구분
   "10개는 기술기업 / 3개만 아님"

③ 상식 자극 ("근데 이상한 게 있어")
   "오프라인 유통이 왜 여기 있지?"

④ 상식이 왜 틀렸는지 예고
   "대체 왜 이렇게 비싸냐, 오늘 얘기해볼 건데"
```

**추출 공식:**
> `충격 숫자` → `이분법 구도` → `상식 자극` → `"근데 왜?"` 예고

내 채널 이식 예시:
> "인간이 하루에 내리는 결정이 35,000개입니다. 이 중 진짜 의식해서 내리는 결정은 딱 3%예요. 나머지 97%는 무의식이 대신 내리는데, 근데 그 무의식이 왜 항상 나쁜 쪽으로 작동하는지 오늘 얘기해볼 겁니다."

## 3. 서사 구조 쪼개기

### 실제 흐름

```
[위기]     2015년 아마존이 월마트 시총 추월
              ↓
[심화]     워런 버핏도 주식 팔았다
              ↓
[반전]     그 이후 10년 우상향
              ↓
[해설]     왜? → 외부 요인 + 내부 요인
```

**핵심 패턴:** 권위자의 실수 → 반전 → 이유 해설

"워런 버핏도 팔았다"는 게 왜 강력한가:
- 버핏이 틀렸다 = 상식이 틀렸다 = 내가 모르는 뭔가가 있다
- 권위자를 끌어내릴수록 시청자의 궁금증이 증폭됨

**추출 공식:**
> `[전문가/상식의 판단]` → `[그게 틀렸다는 반전]` → `[진짜 이유]`

내 채널 이식 예시:
> "심리학자들은 의지력이 약해서 습관을 못 만든다고 했습니다. 틀렸습니다. 최신 연구에서 의지력은 아예 존재하지 않는다는 결과가 나왔어요. 그럼 습관은 어떻게 만드는 건가?"

## 4. 정보 전달 방식 쪼개기

### 패턴 1: 한국 사례로 번역

원본 개념 → 한국 사례 매핑

| 원본 | 번역 |
|------|------|
| 월마트 PB 그레이트 밸류 | 노브랜드·곰곰·탐사 |
| 월마트+ 멤버십 | 쿠팡 로켓배송 |
| 월마트 = 이마트+쿠팡 합체 | 한국인이 바로 이해하는 구조 |

**공식:** 낯선 개념 → 청중이 이미 아는 것으로 1:1 번역

### 패턴 2: 숫자를 체감 단위로 변환

| 원본 숫자 | 변환 방식 |
|-----------|-----------|
| 210만 명 고용 | "군대보다 많은 수준" |
| 22위 밖 국가보다 매출 많음 | "160개 나라가 월마트보다 못 벌어" |
| 바나나 40억 개 판매 | 추상적 매출 → 친숙한 물건으로 |

**공식:** 큰 숫자 → 청중이 감각적으로 느낄 수 있는 단위로 변환

### 패턴 3: 구조를 먼저 예고

> "두 가지로 볼 수 있을 것 같아요. 하나는 외부 상황이 유리해진 측면, 또 하나는 월마트가 경쟁력을 키운 측면"

→ 목차를 먼저 말함 → 시청자가 이탈 안 하고 기다림

**공식:** 본론 들어가기 전 "~가지로 나눠볼게요" 예고

## 5. 감정 설계 쪼개기

이 영상에서 시청자가 느끼는 감정의 흐름:

```
① 놀라움  — "1조 달러짜리 오프라인 마트가 있다고?"
② 공감    — "오프라인 망하는 거 나도 알지"
③ 의심    — "근데 왜 잘됐대?"
④ 통쾌함  — "버핏도 틀렸네"
⑤ 이해    — "아 이런 구조구나"
⑥ 현실감  — "이미 늦었대, 그냥 공부로 봐야겠다"
```

**공식:** 놀라움 → 공감 → 의문 → 반전 → 이해 → 현실 착지

마지막 "이미 늦었다"의 기능:
- 투자 권유 채널 아님을 선언 → 신뢰도 UP
- "그럼 뭐 배웠지?" 복기 욕구 자극 → 저장·공유율 UP

## 6. 비유 패턴 쪼개기

이 영상에서 쓴 비유 목록:

| 비유 | 원리 |
|------|------|
| 월마트 고용 = 도시 국가 | 추상적 규모 → 구체적 실체 |
| 월마트 = 왕, 공급사 = 신하, 소비자 = 백성 | 권력 관계를 역할극으로 시각화 |
| 우유 반복 구매 → 라겐 효과 | 행동 원리를 일상 사례로 설명 |
| 배민 광고비 = 월마트 검색 광고 | 낯선 것을 이미 아는 것으로 연결 |

**공식:** 새로운 개념을 설명할 때 반드시 청중이 이미 경험한 것과 1:1 매핑

## 추출한 공식 총정리

### 공식 1 — 제목
`[의외의 주체]` + `[공격적 동사]` + `[숨겨진 이유]`

### 공식 2 — 오프닝
`충격 숫자` → `이분법` → `상식 자극` → `"왜?" 예고`

### 공식 3 — 서사
`권위자/상식의 판단` → `반전` → `진짜 이유 해설`

### 공식 4 — 정보 전달
- 낯선 개념 → 청중의 기존 경험으로 번역
- 큰 숫자 → 감각적 단위로 변환
- 본론 전 구조 예고

### 공식 5 — 감정 흐름
`놀라움` → `공감` → `의문` → `반전` → `이해` → `현실 착지`

### 공식 6 — 마무리
자기 부정 선언 ("이게 답이 아님", "이미 늦었다") → 신뢰도 + 저장욕 자극
