## 핵심 전제: 이건 기존 워크플로우와 다른 구조다

북타쿠 스타일 카드뉴스의 재료는 **연구 결과**다.
전쟁사 카드뉴스의 재료는 **실제로 일어난 일**이다.

이 차이가 워크플로우 전체를 바꾼다.

```
북타쿠: 연구 결과 → 뇌과학 설명 → 독자 레이블
전쟁사: 역사 사건 → 인간 본능 분석 → 현대 연결
```

권위의 원천이 다르다.
"하버드 연구"가 아니라 "실제로 이런 일이 있었다"가 신뢰를 만든다.
이게 오히려 더 강할 수 있다. 반박이 불가능하기 때문에.

---

## 이 콘텐츠의 진짜 차별점

**표면:** 전쟁사 이야기
**실제:** 극단 상황에서만 드러나는 인간 본능의 증거

전쟁터는 인간 심리 실험실이다.
- 도덕 브레이크가 해제된 상태
- 생존 본능이 날 것으로 나오는 상태
- 집단 심리가 극대화된 상태

이걸 분석하면 평시 일상의 "약한 버전"을 설명하는 도구가 된다.

예:
```
전쟁터: 항복한 적을 살려줌 → 다음에 내가 잡혔을 때 살아남기 위해
일상:   경쟁자를 봐줌 → 언젠가 내가 도움받을 때를 대비해
원리:   호혜적 이타주의 (트리버스)
```

독자는 전쟁 얘기를 보는 게 아니라
**자신의 행동 원리를 역사로 확인**하는 것이다.

## 기존 워크플로우 vs 전쟁사 워크플로우

| 단계 | 기존 (북타쿠 스타일) | 전쟁사 특화 |
|---|---|---|
| 재료 수집 | 레퍼런스 스크린샷 수집 | 매트릭스에서 주제 선정 |
| 사실 검증 | 연구 논문 인용 | **역사 사실 팩트체크** (필수) |
| 카피 공식 | 레이블 + 상식위반 + 숫자 | 역사 사건 + 반전 + 현대 연결 |
| 슬라이드 구조 | 충격→이유→실행→책 | 사건→본능분석→진화→현대→교훈 |
| 이미지 소스 | 드라마 스틸컷 | 전쟁 영화 스틸컷 + 역사화 |
| 캡션 추가 요소 | 연구 기관 권위 | 역사적 구체성 (날짜, 인원, 장소) |
| 시리즈 구조 | 독립 게시물 위주 | 매트릭스 기반 시리즈 |

## 전쟁사 콘텐츠 매트릭스

이 매트릭스가 핵심 자산이다. 주제 고갈 없음.

```
         │진화심리│행동경제│뇌과학 │게임이론│인지편향│평판이론
─────────┼────────┼────────┼───────┼────────┼────────┼────────
포에니   │  ●     │  ●     │       │  ●     │        │
십자군   │        │        │  ●    │        │  ●     │
몽골     │  ●     │        │       │  ●     │        │  ●
1차대전  │  ●     │  ●     │       │  ●     │        │
2차대전  │        │        │  ●    │        │  ●     │
명량해전 │  ●     │        │  ●    │        │        │
스파르타 │  ●     │        │  ●    │        │        │
남북전쟁 │  ●     │        │  ●    │        │  ●     │
```

● 표시된 칸이 1편. 현재 파일에 나온 아이디어만으로도 30편 이상.
매트릭스를 확장하면 100편+.

**운영 방식:**
- 매주 파일에서 칸 하나 선택
- 같은 전쟁을 다른 도구로 2-3편 시리즈화 가능
- "포에니 전쟁 시리즈 3부작" 같은 묶음 기획 가능

## 전쟁사 특화 워크플로우 (단계별)

### STEP 1 — 주제 선정 (5분)

매트릭스에서 전쟁 × 도구 조합 하나 선택.

선택 기준:
```
□ 현대 일상과 연결이 명확한가
□ 카피 한 줄이 바로 떠오르는가
□ 역사 사실이 충분히 알려진 사건인가 (검증 부담 낮음)
□ 독자가 "나도 그런 적 있다"고 느낄 수 있는가
```

---

### STEP 2 — 역사 사실 검증 (10분) ← 기존 워크플로우에 없는 단계

**이 단계를 건너뛰면 신뢰가 무너진다.**

Claude Code에 요청:
```
[전쟁/사건명]에서 [핵심 행동/사실]에 대해
- 날짜, 인원, 장소 정확하게
- 잘못 알려진 통설이 있다면 수정
- 인용 가능한 출처 (역사학자, 논문, 1차 사료)
팩트체크해줘. 불확실한 부분은 명시해줘.
```

검증 후 확인 항목:
```
□ 날짜/인원 수치가 사실인가
□ 과장된 통설을 그대로 쓰는 건 아닌가
□ 불확실한 부분은 "~로 알려져 있다" 처리했는가
```

---

### STEP 3 — 카피 생성 (10분)

**전쟁사 카피 3가지 공식:**

**공식 A — 반전형 (억울한 사람 구제)**
```
[잔인해 보이는 행동], 사실은 [본능]이었다
예: "적을 다 죽인 몽골군, 사실 계산이 제일 인간적이었다"
```

**공식 B — 충격 수치형 (권위)**
```
[역사적 구체성] + [예상 밖 결과]
예: "8만 명 죽고도 협상 거부한 나라, 결국 이긴 이유"
```

**공식 C — 현대 연결형 (정체성 버튼)**
```
[전쟁터 행동] = [일상 행동]
예: "전쟁터에서 총 안 쏜 군인 80%, 지금 당신도 하고 있다"
```

조건:
```
- 역사적 구체성이 숫자 역할을 대신함 (날짜, 인원, 장소)
- 단정적 어조
- 현대 연결이 바로 보여야 함
- 18자 이하
```

---

### STEP 4 — 슬라이드 구성 (10분)

**전쟁사 6장 구조 (기존 구조 변형):**

```
1장 — 카피 (역사 사건 후킹)
       "어제까지 총 쏘던 적에게 초콜릿을 건넨 날"

2장 — 역사 사실 (2-3줄, 간결하게)
       언제 / 어디서 / 무슨 일

3장 — 심리 분석: 왜 이런 행동이 나왔나
       진화심리 / 뇌과학 / 행동경제 도구 1개 적용

4장 — 진화 원리: 왜 이 본능이 만들어졌나
       10만 년 전 부족 환경으로 설명

5장 — 현대 연결: 지금 우리 일상에서 똑같이 나타나는 것
       직장 / 관계 / SNS 등 구체적 예시

6장 — 교훈/실행법 (저장 유발)
       이 본능을 알면 뭘 다르게 할 수 있는가
```

---

### STEP 5 — 캡션 생성 (10분)

**전쟁사 캡션 구조 (기존 + 역사 섹션 추가):**

```
오프닝: 역사 사건 한 줄 + 반전 예고
⠀
🏛️ [역사 사실]
날짜, 인원, 장소, 핵심 행동 — 구체적으로

🎓 [연구/이론]
이 행동을 설명하는 심리학/진화심리학 연구

🧠 [책]
이 본능의 진화적 기원 설명

현대 연결:
"지금 우리 일상에서 이 본능이 이렇게 나타납니다"

결론:
"결국 이건 의지의 문제가 아니라 [본능]의 설계입니다"

실행법:
이 본능을 인식했을 때 할 수 있는 것 1가지

📚 책 추천

📚 질문 CTA

📌 3줄 행동 CTA
```

---

### STEP 6 — 이미지 소싱 (15분)

**전쟁사 이미지 소스 (기존과 다름):**

우선순위:
```
1순위: 전쟁 영화 스틸컷
       (300, 퓨리, 덩케르크, 미드웨이, 한산, 명량, 남한산성...)
2순위: 역사화/그림 (저작권 없음)
3순위: 실제 역사 사진 (저작권 만료 확인 필수)
4순위: AI 생성 이미지 (GPT)
```

톤:
- 전쟁 장면보다 **감정이 드러나는 인간 클로즈업**
- 잔인함보다 **인간적 순간** (항복, 눈물, 결의)
- 어두운 배경 + 강한 명암 대비

## 콘텐츠 발행 리듬 제안

```

월요일 — 전쟁사 카드뉴스 (시리즈 편)

```

- 다른 심리 계정과 차별화
- 역사 관심 독자층 추가 유입
- "이 계정은 뭔가 다르다" 인식 형성

## 병목 포인트와 해결책

### 병목 1 — 역사 사실 오류 리스크

전쟁사는 틀리면 신뢰가 바로 무너진다.
댓글로 잡힌다.

해결:
- STEP 2 팩트체크를 절대 건너뛰지 않는다
- 불확실한 수치는 "약 ~명", "~로 알려진" 처리
- 논란이 있는 사건(위안부, 일제 관련)은 별도 검토

### 병목 2 — 전쟁 이미지 조달

드라마 스틸컷보다 적합한 이미지 찾기가 더 어려움.

해결:
- 전쟁 영화 스틸컷 풀 미리 구축
  - 카테고리: 고대전 / 중세전 / 근대전 / 현대전
  - `029_Collections/전쟁사 이미지 레퍼런스/` 폴더 생성
- 역사화는 저작권 없어서 바로 활용 가능

### 병목 3 — 캡션이 길어짐

역사 배경 설명이 추가되어 기존보다 길어짐.

해결:
- 역사 사실 섹션은 3줄 이내로 압축
- 독자가 이미 아는 유명 사건 우선 선택
  (역사 설명 최소화 가능)

## 시리즈 기획 예시 (즉시 실행 가능한 것)

**단기 3부작: "전쟁터에서 드러난 인간성"**
```
1편 — "어제까지 총 쏘던 적에게 초콜릿을 건넨 날"
      (1차대전 크리스마스 휴전 × 호혜적 이타주의)

2편 — "총알이 빗발치는데 총 안 쏜 군인이 80%"
      (남북전쟁 × 동족 살해 저항 본능)

3편 — "적을 학살하고 울었던 십자군"
      (십자군 × 도덕적 면허 + 신성화된 폭력)
```

**중기 5부작: "이기는 심리학"**
```
1편 — 몽골군의 계산된 잔혹함 × 게임이론
2편 — 한니발에게 져도 협상 안 한 로마 × 손실회피
3편 — 13척으로 133척 이긴 이순신 × 편도체+필사
4편 — "설마 그러겠어" 하다 5분에 망한 일본 × 확증편향
5편 — 스파르타의 인간 개조 프로젝트 × 뇌과학
```

## 카피 10개 (즉시 사용)

기존 파일 아이디어 기반, 개선 공식 적용:

```
1. 어제까지 총 쏘던 적에게 초콜릿을 건넨 이유
2. 8만 명 죽고도 협상 거부한 나라, 결국 이겼다
3. 총알 빗발치는데 총 안 쏜 군인이 80%였다
4. 몽골군이 한 마을만 다 죽인 진짜 이유
5. 학살하고 울었다, 가장 잔인한 군인의 비밀
6. 스파르타가 아기를 지옥에 던진 이유
7. 도망치면 살고 싸우면 죽는데, 왜 싸웠나
8. 절대 포기 안 하는 사람, 로마가 증명했다
9. 제복 입으면 사람이 달라지는 이유
10. 전쟁터에서 가장 먼저 무너지는 건 시간 감각이다
```
