## 두 분석 비교 — 갭과 개선점

Claude 분석 (`북타쿠-카드뉴스-패턴분석.md`) vs GPT 분석 (사용자 공유)

---

## Claude가 놓친 것 (GPT가 잡아낸 것)

### 1. 권위 레이어 — 독립 메커니즘으로 분석 안 함

내 분석에서 "구체적 숫자"와 "출처"를 묶어서 봤는데,
GPT는 이걸 **별개의 심리 트리거**로 분리했다.

```
하버드 85년 연구 / MIT / 노벨상 / 세계 100인
→ "내 의견이 아님" → "전문가가 말함" → 클릭 비용 하락
```

권위 기관명 자체가 독립 후킹 수단임. 숫자와 다른 층위.

**실전 적용:** 카피에 숫자뿐 아니라 "하버드", "MIT" 같은 기관명을 별도로 넣는 전략이 필요.

---

### 2. 절대 표현 단어 목록 — 뭉뚱그렸음

내 분석: "단정적 어조"

GPT 분석: **단어 목록으로 제시**
```
가장 / 절대 / 평생 / 상위 10% / 충격적 / 반드시
```

내가 "단정적으로 써줘"라고 프롬프트에 넣은 것보다
이 단어 목록을 직접 넣는 게 결과물 퀄리티가 다름.

---

### 3. 색상 심리학 — 완전히 누락

내 분석: "흰색 볼드 고딕" 수준

GPT 분석:
```
흰색 → 정보 전달
노랑 → 핵심 강조
빨강 → 공포/위험 신호
```

색상이 심리적 역할을 분담하고 있다는 분석 자체가 없었음.
썸네일 커버 기획에 반영 필요.

---

### 4. 슬라이드 전체 흐름 구조 — 1장만 분석

내 분석: 첫 번째 슬라이드(후킹) 중심

GPT 분석: **슬라이드 플로우 전체**
```
충격 → 왜? → 이유 5가지 → 실행법 → 책 추천
```

저장을 유발하는 건 1장이 아니라 이 전체 구조.
"나중에 따라해야지"를 만드는 건 실행법+책 추천 슬라이드.

---

### 5. 공포 분류 — 2개 → 4개

내 분석: "불안과 관계" 2가지

GPT 분석:
```
건강 공포 — 수면, 비만, 노화, 우울, ADHD
관계 공포 — 이별, 연애, 친구, 애착
성공 공포 — 루틴, 지능, 성공 확률
심리 공포 — 멘헤라, 트라우마, 비교, 자존감
```

주제 선정 시 이 4개 버킷으로 생각해야 조회수 예측이 가능해짐.

---

### 6. 핵심 인사이트 깊이 — 한 층 얕았음

내 분석: "책 얘기 안 하고 뇌과학으로 포장"

GPT 분석: **"새로운 자아를 판다"**
```
독자가 책을 저장하는 진짜 이유:
"나도 더 똑똑해지고 싶다"
"나도 성공하고 싶다"
"나도 우울에서 벗어나고 싶다"
→ 책은 변신 도구
```

내 분석은 포장 방식을 봤고, GPT 분석은 독자의 진짜 욕구를 봤다.
한 층 더 깊은 인사이트.

## Claude가 더 잘 잡은 것

- **15개 카피 전수 분석** — GPT는 일부 예시만, 나는 전량 패턴화
- **정체성 버튼 6가지 유형** — GPT보다 세분화
- **댓글 스크린샷 오버레이** — 사회적 증거 메커니즘 발견
- **이미지 소스 전략** — 드라마 스틸컷 활용 이유까지 설명
- **교차참조** — 위키 개념과 연결

## 합치면 나오는 완전한 공식

```
[독자의 공포 버킷 선택]
    건강 / 관계 / 성공 / 심리
        ↓
[정체성 레이블 + 절대 표현]
    "첫째 딸" / "35~45세" / "상위 10%" / "절대"
        ↓
[권위 기관 + 숫자]
    "하버드" / "MIT" / "3배" / "40%"
        ↓
[슬라이드 플로우: 충격 → 이유 → 실행 → 책]
        ↓
[색상: 흰(정보) + 노랑(핵심) + 빨강(공포)]
        ↓
좋아요 + 저장 + 공유
```

## 실전 반영 포인트

**카피 생성 프롬프트에 추가할 것:**
```
- "가장/절대/상위 10%/반드시/평생" 중 1개 포함
- 권위 기관명(하버드, MIT, 연구팀) 포함 가능하면 포함
- 공포 버킷 4개 중 어디에 속하는지 명시하고 생성
```

**썸네일 커버 기획에 추가할 것:**
```
- 색상 역할 명시: 흰색(정보) / 노랑(핵심) / 빨강(공포·강조)
```

**캡션 구조에 추가할 것:**
```
- 실행법 슬라이드 기획 (저장 유발 핵심)
- 책 추천을 마지막에 배치 (변신 도구로서 포지셔닝)
```
