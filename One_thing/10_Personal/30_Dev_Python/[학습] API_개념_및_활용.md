모르는 용어 
- Auth 
	- 로그인이 필요한 경우 `apiKey`로 표시 
	- 로그인이 필요 없는 경우 No로 표시 
- HTTPS 
	- 보안 연결 제공 여부 
- CORS 
	- HTML 페이지 내 요청에 따른 오류 여부 
		- YES -> HTML에서의 요청은 거부 
		- NO -> HTML에서 요청하기 가능 

 AJAX 를 이용하면 페이지는 그대로 두고 정보만 쏙 빼올 수 있음. 


AJAX 요청법
1. XMLHttpRequest (옛날방식)
2. jQquery Ajax (옛날방식)
3. fetch - 최신 표준
3가지 모두 정보를 받아와서 요청하는 방식이다.
