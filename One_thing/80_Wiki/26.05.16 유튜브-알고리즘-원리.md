> **핵심 Takeaway (콘텐츠 방향성 관점)**
> 유튜브는 창작자의 무대가 아니라 구글의 광고 판매처다. 이 한 줄을 깔면 알고리즘의 모든 결정이 이해된다. 콘텐츠 비즈니스 설계의 출발점.

## 유튜브-알고리즘-원리

### 대전제: 구글은 광고 회사다

"내 영상이 어떻게 구글에게 돈이 되는가"를 기준으로 모든 결정을 하면 흔들리지 않는다.

구조적 결론:
- 8분 이상 → 미드롤 광고 → 노출 1.5~3배
- 길수록 노출 증가 (한국 탑 50~60분, 해외 탑 2~3시간)
- 조회수가 구독자의 100%면 구글애즈 필요 없음 → 그래서 일반 채널 기준 30%가 매우 양호, 10%가 건강한 편

### 알고리즘은 두 가지만 알면 된다

1. **탐색(Exploration)**: 시청자 피드에 노출되는 것 — 압도적으로 중요
2. **검색(Search)**: 직접 검색 — 실제로 검색하는 사람은 생각보다 적다 (특히 어린이·중장년층)

게임의 전부: "어떻게 시청자의 탐색 피드에 뜨게 할 것인가"

### MrBeast 4단계 공식 (평균 13분 37초 기준)

| 구간 | 역할 |
|------|------|
| 0~1분 | 조명·나레이션·드론 총동원, 가장 강렬하게 (6천만 클릭 → 1분 안에 2,100만 이탈) |
| 1~3분 | 몰입형 스토리. 일주일 서바이벌을 3분에 압축 |
| 3분 지점 | 재참여 포인트 — 다른 유튜버가 못 할 스펙터클 한 방 |
| 3~6분 | 가장 자극적·단순한 콘텐츠 |
| 6~13분 | 광고 배치. **끝난다는 신호를 절대 주지 말 것** |

### 오프닝에서 절대 하지 말아야 할 3가지
1. 자기소개
2. 지난 화 하이라이트 / 이번 화 예고
3. 채널 로고 인트로

### 4단계 팬덤 구조 (케빈 켈리, 1000 트루팬, 2008)

시청자 → 구독자 → 팬 → 트루팬(고객)

구독자는 팬이 아니다. 몇 번 안 보면 피드에도 안 뜬다. 팬덤은 영상 보는 순간이 아니라 영상 끝나고 대화하는 시간에 깊어진다.

### TV 시청 시대와 콘텐츠 전략

닐슨 기준 2024.12: 유튜브 11.1% / 넷플릭스 8.5% (점차 확대 중)

- TV = 배속 불가 → 시청 시간이 절대값으로 높아짐
- 비디오 팟캐스트: 설거지·운전 중 광고 스킵 불가 → 구조적으로 광고 효율 최고
- 린백 콘텐츠(브이로그·캠핑·여행)가 TV에서 강함

### 발견의 3원칙 (이연복 셰프 사례)

1. **대중성**: 마이너 재료보다 누구나 아는 재료
2. **시의성**: 치킨 논란 때는 치킨 레시피
3. **반복성**: 노출이 4~5일이라 같은 주제도 새로 올리면 새 영상

→ 이연복 채널 한 줄 요약: "중식 레시피 채널"이 아닌 **"중식 재료로 만드는 집밥 채널"**

### 썸네일 전쟁

- 노출이 4~5일 → 그 안에 3~4번 교체가 기본
- 마키스 브라운리: 썸네일 1장에 4~5시간 투자
- 썸네일 제작 비중 = 본편과 50:50

### 멀티포맷 루프

```
숏폼으로 신규 유입
    ↓
롱폼으로 시청 시간·팬덤 구축
    ↓
게시물·라이브로 관계 지속
    ↓
유튜브 쇼핑으로 매출 전환
```
