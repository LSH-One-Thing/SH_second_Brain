# 위키 전체 목차

> **분석 도구 아카이브** — 행동경제학 × 심리학 × 진화심리학 × 신경과학 × 철학은 **렌즈**다. 채널의 메인 주제가 아니다.
> 채널 주제는 "거시 사건·사회 현상을 인간 본능으로 해석하고 개인 삶으로 착지"다. 도구가 주제를 결정하지 않는다.

---

   ↙      ↓       ↘

        ↓

        ↓
[집단정체성-내외집단] ←── [에릭호퍼-좌절한-이상주의자]
        ↓
인문/독서 기반
[빅터프랭클-의미를-찾아서] ←── [데미안-알깨기]  ←── 고통을 통한 성장
        ↓                               ↓
[팩트풀니스-미디어편향] ←── [손실회피] ←── 미디어 경제 설계
        ↓

        ↓

## 의외의 연결 지도 (Cross-Domain Connections)

가장 충격적인 교차 연결 — 다른 도메인에서 동일 원리가 독립적으로 발견된 것들:

| 연결 | 핵심 통찰 |
|---|---|
| [[전략적-대충-역설]] ↔ [[개미 투자자가 실패하는 이유]] | 심리학 채널과 투자 채널이 완전히 다른 경로로 "아무것도 안 하는 것이 최고"라는 동일 결론 도달 |
| [[행복최적화-역설]] ↔ [[전략적-대충-역설]] ↔ [[개미 투자자가 실패하는 이유]] | 행복 심리학 + 투자 + 뇌과학 세 도메인이 독립적으로 "강도 추구 → 역효과"라는 동일 구조 발견 |
| [[가까운 사람에게 화를 더 잘 내는 이유]] ↔ [[에릭호퍼-좌절한-이상주의자]] | 가족에 대한 분노 ↔ 광신자의 내부 배신자 혐오: 기대치가 높은 대상에게 실망이 폭발하는 동일 구조 |
| [[개미 투자자가 실패하는 이유]] ↔ [[더닝-크루거-효과]] | 가장 많이 공부한 20대가 꼴찌: 지식이 자신감을 높이지만 구조적 불리함을 이기지 못한다 |
| [[패권경쟁-심리학]] ↔ [[가까운 사람에게 화를 더 잘 내는 이유]] | 국가 간 동맹 배신 ↔ 가족 폭발: 기대치와 애착이 클수록 배신의 충격이 크다 |
| [[전략적-대충-역설]] ↔ [[편도체-강탈]] | 완벽주의 강박 = 편도체 강탈: 뇌에서 동일하게 처리됨 |
| [[에릭호퍼-좌절한-이상주의자]] ↔ [[더닝-크루거-효과]] | 좌절한 이상주의자 = 더닝-크루거(무지의 산) + 자의식 방어. "내가 못 나서가 아니라 시스템이 나빠서" = 능력 부족을 인식 못한 채 외부 귀인하는 구조 |
| [[신뢰-프리미엄-비탄력성]] ↔ [[집단정체성-내외집단]] | 충성 구독자 = 내집단 형성. 브랜드 충성도와 종교 충성도와 팬덤이 같은 원리 |
| **[[자청-시스템1-원시인뇌]] ↔ [[클루지-진화의-유산]] ↔ [[편도체-강탈]]** | **3자 수렴 (충격):** 카네만(심리학), 자청(사업), 마커스(신경과학)가 완전히 독립적으로 같은 결론에 도달 — 시스템1 = 클루지 = 편도체 강탈 = 원시인 뇌 |
| **[[자청-본능분석-반박제거]] ↔ [[에릭호퍼-좌절한-이상주의자]]** | **마케터와 혁명가의 동일 공식:** 선동 = 마케팅. "당신이 나쁜 게 아니야"는 본능 파악 + 반박 제거의 공식 |
| **[[자청-본능분석-반박제거]] ↔ [[자의식-방어기제]] ↔ [[확증편향]]** | 가치 사다리의 심리학적 기반: 자의식 방어 + 확증편향이 마케팅 구조를 지탱한다 |
| **[[자청-UIS-창조방법론]] ↔ [[직업탐색-동사법]]** | 불편함이 사업 씨앗 = 불편함이 직업 씨앗. 같은 방법으로 찾아진다 |
