> **핵심 Takeaway (콘텐츠 방향성 관점)**
> 카드뉴스 제작의 병목은 반복 작업이다. Claude Code(분석·기획·캡션) → Gemini(스프레드시트) → Canva(대량 제작) 파이프라인으로 기획부터 제작까지 자동화한다. GPT는 이미지 생성 전용.

## 인스타 카드뉴스 AI 제작 워크플로우

**툴 역할 분담**

| 도구 | 역할 |
|---|---|
| Obsidian + Claude Code | 레퍼런스 분석, 카피 기획, 캡션 생성 |
| Gemini | 카피 + 캡션 → 스프레드시트 변환 |
| Canva | CSV 가져오기로 대량 제작 |
| GPT | 카드 이미지 생성 (선택) |

---

## 전체 파이프라인

```
레퍼런스 수집
    ↓
Claude Code — 패턴 분석 + 카피 30개 생성
    ↓
3-4개 선별
    ↓
Claude Code — 캡션 생성 (선별 카피 입력 → 래퍼런스 접근)
    ↓
Gemini — 카피 + 캡션 스프레드시트 변환
    ↓
CSV 다운로드
    ↓
Canva — CSV 가져오기 → 대량 제작
    ↓
발행 → 성과 피드백 루프
```

---

## 단계 1 — 레퍼런스 수집 (30분/주)

잘 터진 카드뉴스 스크린샷 15-30개 수집.

- 좋아요 1000+ 또는 저장 100개+ 기준

---

## 단계 2 — Claude Code 패턴 분석 (20분)

스크린샷을 Obsidian에 첨부 후 Claude Code에게 분석 요청.

```
이 카드뉴스들이 왜 터졌는지 심층 추론해줘.
분석 항목: 카피 구조, 후킹 포인트, 상식 위반 패턴, 정체성 버튼 등 알아서 추론
```

---

## 단계 3 — 카피 30개 생성 (15분)

분석 결과 기반으로 Claude Code에게 카피 생성 요청.

```
위 분석 기반으로 [채널 주제] 카드뉴스 카피 30개 출력.
조건: 번호만, 단정적 어조, 숫자 포함, 18자 이하, 추상어→구체어
```

---

## 단계 4 — 선별 (15분)

체크리스트 — 3개 이상이면 선발:

```
□ 상식 위반 (기존 믿음 뒤집기)
□ 정체성 버튼 ("나 해당되는데?" 자극)
□ 구체적 숫자
□ 단정적 어조
□ 채널 주제 일치
```

---

## 단계 4-1 — 캡션 생성 (카피당 10분)

**트리거:** 사용자가 선별된 카피를 Claude Code에게 제공

**Claude Code 실행 순서:**
1. `029_Collections/카드뉴스 래퍼런스/26.05.21/북타쿠/캡션 래퍼런스/` 접근
2. 래퍼런스 캡션 패턴 분석
3. 입력된 카피 기반 캡션 생성

**사용자 입력 형식:**
```
카피: [선별된 카피 텍스트]
캡션 생성해줘.
```

---

### 캡션 구조 공식 (래퍼런스 기반)

북타쿠 캡션 9개 분석으로 추출한 반복 구조:

```
[오프닝 — 2-3줄 후킹]
기존 편견 또는 대화체로 시작 → "하지만 데이터는 정반대였습니다."

🎓 [연구기관/학자] 연구에 따르면
[구체적 수치 + 논문 출처]

🧠 『[주제에 맞는 책]』에 따르면
[책의 핵심 주장 1-2줄]

[책 관점으로 본 원인 분석]

[현대 환경과의 연결 또는 충돌]

결국 [카피 주제]는
의지의 문제가 아니라
설계와 환경의 [일치/충돌]입니다.

📚 여러분은 [카피 주제와 연결된 질문]?

📌 도움이 되셨다면 좋아요
📌 나중에 다시 보려면 저장
📌 계속 [채널 테마] 콘텐츠를 받고 싶다면 팔로우
```

**책 선택 기준 — 카피 주제에 따라 달라짐**

| 카피 주제 | 적합한 책 예시 |
|---|---|
| 진화·본능·뇌 설계 | 『클루지』게리 마커스, 『사피엔스』유발 하라리 |
| 감정·심리·우울 | 『우울할 땐 뇌 과학』앨릭스 코브, 『감정은 어떻게 만들어지는가』리사 배럿 |
| 습관·의지·성과 | 『아주 작은 습관의 힘』제임스 클리어, 『의지력의 재발견』켈리 맥고니걸 |
| 관계·사랑·사회성 | 『관계의 과학』니컬러스 크리스태키스, 『사랑의 기술』에리히 프롬 |
| 인지편향·의사결정 | 『생각에 관한 생각』대니얼 카너먼, 『넛지』탈러·선스타인 |
| 성공·끈기·성장 | 『그릿』앤절라 더크워스, 『마인드셋』캐럴 드웩 |

→ **검증 원칙:** 실제 출판된 책, 내용과 주장이 카피 맥락에 사실적으로 맞는 경우만 인용. 불확실하면 🎓 연구 인용으로 대체.

---

### 캡션 생성 프롬프트 (Claude Code용)

```
카피: [카피 텍스트]

029_Collections/카드뉴스 래퍼런스/26.05.21/북타쿠/캡션 래퍼런스/ 폴더를 읽고
래퍼런스 캡션 구조를 기반으로 이 카피에 맞는 캡션을 작성해줘.

조건:
- 오프닝: 기존 편견 제시 → "하지만 데이터는 정반대였습니다" 반전 구조
- 🎓 실제 연구/논문 인용 (수치 포함, 검증된 사실만)
- 🧠 카피 주제에 가장 잘 맞는 책 1권 선택 (진화·뇌 주제 → 클루지류, 감정 → 뇌과학류, 습관 → 행동심리류 등)
- 결론: "의지의 문제가 아니라 설계와 환경의 [일치/충돌]입니다"
- 마지막: 📚 질문 + 📌 3줄 CTA
```

---

## 단계 5 — Gemini 스프레드시트 변환 (10분)

선별된 카피 + 생성된 캡션을 Gemini에 붙여넣기.

```
아래 카드뉴스 기획을 Canva CSV 가져오기용 스프레드시트로 만들어줘.
컬럼: 슬라이드번호 | 카피 | 서브텍스트 | CTA | 캡션
[카피 + 캡션 붙여넣기]
```

→ 스프레드시트 완성 후 **CSV로 다운로드**

---

## 썸네일 커버 기획 표준

캡션 파일 생성 시 항상 포함:

```
## 썸네일 커버 기획

**이미지 사이즈:** 1080 × 1350 픽셀 (4:5)

**콘셉트:** [한 줄 방향성]

**이미지 방향:**
- [옵션 1]
- [옵션 2]
- 톤: [색감/분위기]
```

---

## 단계 6 — Canva 대량 제작 (30분)

1. Canva에서 카드뉴스 템플릿 선택
2. `앱 → 대량 제작 → CSV 가져오기`
3. CSV 연결 → 자동 생성
4. (선택) GPT로 슬라이드 이미지 생성 후 삽입

---

## 단계 7 — 피드백 루프 (월 1회)

저장 수 상위 3개 → Claude Code 분석 → 패턴 추가
저조한 게시물 댓글 → Claude Code 분석 → 개선 플랜

---
