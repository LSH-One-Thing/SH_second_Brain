## 카피 구조 — 3가지 패턴

**패턴 A — 타겟 명시 + 미래 손실 경고**

| 예시 | 구조 |
|---|---|
| "여자 나이 35-45세, 운동 안하면 5년뒤 벌어지는 5가지" | 나이+성별 타겟 / 시간 손실 |
| "가족 중 '첫째 딸'이 우울증 걸릴 확률 가장 높다" | 가족 역할 타겟 / 위험 경고 |
| "지금 20~40대 우울증보다 먼저 오는 '멀티탭 증후군'" | 나이 타겟 / 신조어 공포 |

→ 타겟이 좁을수록 해당하는 사람은 멈춘다. 나만을 위한 정보처럼 느끼기 때문.

**패턴 B — [일상 행동] + [충격적 등가물/수치]**

| 예시 | 충격 포인트 |
|---|---|
| "2시간 이상 앉아있으면, 흡연하는 것과 같다" | 앉아있기 = 흡연 |
| "불 끄고 샤워하면, 수면 호르몬 70% 더 나온다" | 행동 변경으로 70% |
| "수면 6시간 이하, 기억력 40% 떨어진다" | 숫자 충격 |

→ 이미 매일 하고 있는 행동에 과학적 의미 부여. "나도 이러는데?" 반사적 멈춤.

**패턴 C — 억울한 사람 구제 (타겟 + 긍정 재해석)**

| 예시 | 구제 대상 |
|---|---|
| "잠 많다고 놀린 여자친구, 뇌가 2배 더 일한 거였다" | 게으르다 놀림받은 사람 |
| "쓰레기만 만나는 당신에게 알려주는 부처님 말씀 4가지" | 인복 없다 자책하는 사람 |
| "같이 뒷담화해 주는 친구, 뇌는 연인보다 가깝게 느낀다" | 뒷담화를 죄책감 느끼는 사람 |
| "묵묵히 참고 견디는 사람, 성공 가능성 더 높다" | 참는 게 손해라 느끼는 사람 |

→ "사실 나는 나쁜 게 아니었어" 구조. 저장 + 공유 폭발.

## 상식 위반 — 전수 목록

| 독자의 기존 믿음 | 뒤집기 |
|---|---|
| 앉아있는 건 괜찮다 | = 흡연과 같다 |
| 잠 많은 사람은 게으르다 | 뇌가 2배 더 일한 것 |
| 뒷담화는 나쁜 것 | 뇌는 연인보다 친밀하게 인식 |
| 첫째는 든든하다 | 우울증 확률 가장 높다 |
| 요즘 청소년은 행복하다 | 일제 강점기 우울 수준과 동일 |
| 사랑하는 사람 보는 건 그냥 좋은 것 | 스트레스 수치 수치로 낮아짐 |

## 정체성 버튼 — 6가지 유형

```
나이/성별:   "여자 나이 35-45세", "첫째 딸", "20~40대"
상황:        "쓰레기만 만나는", "멘헤라", "묵묵히 참는"
억울함:      "잠 많다고 놀린", "뒷담화 친구"
습관:        "2시간 이상 앉아있으면", "수면 6시간 이하"
신조어:      "ADHD", "멀티탭 증후군" (궁금증 → 클릭)
권위:        "오타니", "부처님", "노자"
```

공통점: **독자가 새 정보를 받는 게 아니라, 자기 상황이 레이블화되는 것.**
